## 철학

뇌는 단순한 반복 단위(피질 컬럼)가 쌓여서 무한한 복잡성을 만든다.
우리도 같은 방식으로 간다:
- 가장 작은 단위(뉴런)를 먼저 만든다
- 그게 작동하면, 연결한다
- 연결이 작동하면, 학습을 넣는다
- 학습이 작동하면, 기억을 넣는다
- 기억이 작동하면, 판단을 넣는다
- ...계속 쌓는다

## 로드맵 (레이어 방식)

| 레이어 | 이름 | 목표 | 상태 |
|--------|------|------|------|
| L0 | 뉴런 1개 | 인공 뉴런이 뭔지 눈으로 본다 | ✅ 완료 |
| L1 | 뉴런 여러 개 | 여러 뉴런이 연결되면 뭐가 달라지나 | ✅ 완료 |
| L2 | 학습 | 반복하면 패턴을 기억하는가 | ✅ 완료 |
| L3 | 기억 | 배운 것을 저장하고 꺼내 쓸 수 있는가 | ✅ 완료 |
| L4 | 판단 | 충돌하는 욕구 사이에서 최적 행동 선택 | ✅ 완료 |
| L5 | 감정 | 공포/호기심/만족/분노가 판단을 조절 | ✅ 완료 |
| L6 | 예측 | 다음에 뭐가 올지 미리 예상 | ✅ 완료 |
| L7 | 미정 | 항상성 균형? 사회성? 메타인지? | ⬜ 대기 |

→ L0부터 시작. 이전 레이어가 안정돼야 다음으로 간다.
