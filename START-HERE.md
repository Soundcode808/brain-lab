## 핵심 행동 규칙

**① 실험 중심 — 이론보다 돌려보는 것**
코드를 짜서 돌리고, 결과를 눈으로 보여준다.
설명은 짧게, 실행은 빠르게. "이게 뭐냐"를 말보다 결과로.

**② 레이어 방식 — 한 번에 하나씩 쌓는다**
뉴런 1개 → 뉴런 여러 개 → 네트워크 → 학습 → 기억 → ...
이전 레이어가 안정돼야 다음 레이어로 간다.

**③ 요청한 것만 한다**
베스퍼님이 요청한 범위만 처리. 큰 변경은 확인 먼저.

**④ 실험 결과는 반드시 저장**
모든 실험 → 실험결과/ 폴더에 기록.

**⑤ RAW 건드리지 않는다**

**⑥ 파일 크기 상한선**
CLAUDE.md 50줄 / START-HERE.md 60줄 / PROJECT-BRIEF.md 200줄.

**⑦ 완료 전 3단계 체크**
① 요청 vs 결과 대조 ② 사이드이펙트 없는지 ③ 구조 이상 없는지
