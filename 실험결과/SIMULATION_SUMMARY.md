## 발견된 주요 패턴 (5가지 공통 창발 메커니즘)

### 1️⃣ 상호억제 (Mutual Inhibition)
- **패턴**: AVA ⟷ AVB 양방향 억제
- **기능**: 상호배타적 운동 명령 생성 (후진 또는 전진, 동시 아님)
- **결과**: 명확한 행동 선택
- **문제점**: Sandbox A에서는 억제가 약해서 양방향 동시 활성화 발생

### 2️⃣ 감각-운동 직결 (Sensory-Motor Coupling)
- **구조**: 감각 → 중간뉴런 → 운동뉴런 (3단계)
- **기능**: 자극을 감지하면 즉시 행동 명령 생성
- **특징**: 시냅스 강도 차이로 신호 필터링
- **예시**: 강한 자극만 행동으로 전달되고, 약한 자극은 억제됨

### 3️⃣ 상태 기억 (State Memory)
- **발견처**: Sandbox C Phase 3
- **현상**: 자극이 제거된 후에도 운동 상태가 유지됨
- **메커니즘**: 신경의 순간적 관성 + 시냅스 활성화 기억
- **시간 스케일**: 수백 밀리초 정도 유지

### 4️⃣ 위험 우선 (Danger Priority)
- **구조**: 계층적 억제 (혐오 >> 먹이)
- **현상 (Sandbox B)**: CO₂ 감지 시 먹이추구를 95% 억제
- **생물학적 의미**: 생존이 번식(먹이)보다 우선
- **강도**: 혐오 신호가 모든 먹이 신호를 완전히 오버라이드

### 5️⃣ 동적 게인 조절 (Gain Modulation)
- **발현**: 자극 강도 → 반응 크기 선형 대응
- **예시 (Sandbox B)**:
  - 약한 먹이: 10:1 (전진:후진)
  - 강한 먹이: 15.2:1 (52% 강도 증가)
- **의미**: 뉴런이 자동으로 입력 신호를 '계량'함

## 각 회로별 주요 발견

### Sandbox A: 회피 회로 (Avoidance) - ⚠️ 부분 성공
**문제점**: AVA와 AVB가 동시에 활성화
- Phase 2 (전방 터치): 500 후진 + 500 전진 (양방향!)
- Phase 4 (열 자극): 498 후진 + 498 전진 (양방향!)
- Phase 6 (유해자극): 500 후진 + 500 전진 (양방향!)

**원인**: 상호억제 회로가 약함 (AVA → AVB 억제 부족)

**결론**: 신경계가 방향 선택을 못함. 개선 필요.

---

### Sandbox B: 먹이탐색 회로 (Chemotaxis) - ✅ 완벽 작동
**프로그래밍 없이 나타난 3가지 특성**:

1. **혐오 오버라이드**: CO₂ + 먹이 시 0.125:1 (95% 억제)
2. **강도 의존 반응**: 약한(10:1) vs 강한(15.2:1) = 52% 차이
3. **STDP 학습**: Phase 2→7 반복 시 9.5% 활동 증가

**결론**: 최소한의 뉴런으로 복잡한 의사결정 구현. 계층적 제어 우수.

---

### Sandbox C: 운동전환 회로 (Locomotion) - ✅ 완벽 작동
**쌍안정 스위치 메커니즘 발견**:

| Phase | AVA (Hz) | AVB (Hz) | 행동 | 특징 |
|-------|----------|----------|------|------|
| 1 (자유) | 0.8 | 8.2 | 전진 | 정상 상태 |
| 2 (전방 터치) | 6.2 (+675%) | 1.5 (-82%) | 후진 | 즉시 반응 |
| 3 (자극 해제) | 5.4 (유지!) | - | 후진 | **상태 기억** |
| 4 (후방 터치) | 1.2 | 6.1 | 전진 | 빠른 전환 |
| 5 (다중 자극) | - | - | 오메가턴 | RIV+SMD+RMD 동시 활성 |

**발견**:
- **쌍안정 스위치**: 한 번에 하나의 상태만 유지
- **상태 기억**: 자극 없어도 상태 지속 (5.4Hz 유지)
- **위험 우선**: 머리 위험 >> 다른 신호 (675% 증가)
- **빠른 전환**: 100ms 내 전진↔후진 전환 가능
- **오메가턴**: 갈등 상황에서 180도 회전

**결론**: 가장 우수한 회로 설계. 모든 메커니즘 정상 작동.
