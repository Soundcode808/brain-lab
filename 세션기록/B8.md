## git 충돌 패턴 정리

매 커밋마다 반복되는 문제와 해결책:
- GitHub Actions가 매시간 자동 커밋 → 로컬과 충돌
- 해결: `git fetch origin main && git reset --mixed origin/main && git add [파일] && git commit && git push` 한 줄로 연결
- HEAD.lock 발생 시: `rm .git/HEAD.lock` 먼저 실행
