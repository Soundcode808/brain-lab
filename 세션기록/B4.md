### 2. 사고 메커니즘 A조 탐색 (3팀 동시 투입)

**A조-1: 신경과학 이론**
- Global Workspace Theory (Baars, Dehaene): 의식 = 정보의 전역 방송
- Predictive Coding (Friston): 뇌는 반응 기계가 아니라 예측 기계
- Free Energy Principle: 예측 오류를 최소화하는 반복 루프 = 사고
- 핵심 정의: 사고 = 예측 모델과 현실 입력 사이 오류를 최소화하는 계층적 순환

**A조-2: 구현 사례**
- Spaun/Nengo: 250만 뉴런, 8가지 인지 과제 수행
- 최소 사고 회로: Brian2로 5천~1만 뉴런 즉시 구현 가능
- 작업기억: 강한 재귀 연결 + 단기 시냅스 증강

**A조-3: 반응 vs 사고 경계**
- 반응: 200ms 미만, 직접 입출력 연결
- 사고: 500ms 이상, 재귀 + NMDA 지속
- 최소 사고 단위 스펙: 뉴런 1만~5만, 재귀 30~50%, NMDA τ>150ms, E/I=4:1

**합성팀: 방향성 분석**
- Path C (병렬 계층) 강력 추천: C. elegans(반응) + 피질 컬럼(사고) 계층 연결
- 검증 기준: 작업기억 + 맥락의존 반응 + 예측 오류 수정

### 5. (원 섹션 4) — 참고용 구버전 탐색 기록

**발견한 문제:**
- 검증 스크립트에서 자극 후 발화율이 400Hz까지 폭주함
- 원인: NMDA(150ms)가 GABA(5ms)보다 30배 오래 지속 → 시간에 따라 흥분 누적
- W_EE=0.03mV로 줄이면 자극 중 32Hz로 안정되지만 자극 후 계속 상승

**발견한 해결 방향:**
- **적응 전류 (Spike-frequency adaptation)** 추가가 필요
  - 발화할 때마다 과분극 전류가 쌓여 자동으로 속도 제한
  - tau_adapt ~ 100ms (AHP: After-HyperPolarization)
  - 실제 피질 뉴런에도 있는 메커니즘
- W_EE=0.06mV + 적응 전류 파라미터 탐색 중 세션 종료

**다음 세션에서 이어야 할 것:**
1. 적응 전류 강도 파인튜닝 → 자극 중 20-50Hz, 자극 후 10-30Hz 안정
2. 안정화된 파라미터로 V1/V2/V3 재검증 (세 조건 모두 유지되는지)
3. 자극 구별 능력 검증 (다른 입력에 다르게 반응하는가)
4. 모듈 연결 테스트 (두 회로 간 정보 전달)
