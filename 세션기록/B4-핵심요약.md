## 2. 안정화 완료

| | 이전(검증만) | 지금(안정화) |
|--|--|--|
| 자극 후 발화율 | 400Hz 폭주 | **21Hz 안정** |
| 자극 중 | 260Hz | **49Hz** |

**폭주 원인**: NMDA(150ms) vs GABA(5ms) 시정수 차이 → 흥분이 계속 누적  
**해결책 2가지**:
1. **NMDA 포화 모델** — 게이팅 변수 s(0~1), 무한 누적 차단
2. **배경 전류 I_bg=10mV** — 뉴런을 항상 임계점 바로 아래 유지 (시상 입력 시뮬)

---

## 3. 확정된 회로 파라미터

```
N_E = 400 (흥분성)  /  N_I = 100 (억제성)
E→E : NMDA 포화 모델, g_ee = 0.07mV, tau = 150ms
I→E : GABA 빠른 시냅스, tau = 5ms
I_bg = 10mV (배경 전류, E 뉴런 전체)
재귀 연결 p_EE = 0.35
```
