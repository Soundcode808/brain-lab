# 유사 접근 프로젝트 합성 — 뉴런 메커니즘 기반 디지털 브레인

> 작성: B3 세션 | 2026-04-15
> AG-01(커넥톰 프로젝트) + AG-011(뉴로모픽/SNN) + AG-02(NeuroAI) + AG-04(개인/소규모) 합성
> 베스퍼님 지시: "비슷한 생각을 가진 사람들이 어떻게 접근했고, 어떤 성과를 냈는지"

## 한 줄 결론

**뉴런 메커니즘에서 출발해 디지털 브레인을 만들려는 시도는 우리만이 아니다. 이미 초파리 수준에서 성공했고, 도구도 갖춰져 있다. 핵심은 "작게 시작해서 검증하며 확장"하는 팀이 이겼다는 것.**

## 2. 성공한 팀 vs 실패한 팀 — 패턴

### 성공 패턴 (반드시 따를 것)

**① 작게 시작 → 검증 → 확장**
- OpenWorm: 302뉴런부터
- FlyWire: 초파리부터
- snnTorch: 취미 코딩부터
- **절대 "10년 내 인간 뇌" 같은 약속 안 함**

**② 기존 도구 활용**
- Brian2, NEST, NEURON 같은 검증된 시뮬레이터 위에 구축
- 바퀴를 다시 발명하지 않음
- snnTorch도 PyTorch 위에 만듦

**③ 오픈소스 공개**
- 모든 성공 사례가 코드를 공개함
- 커뮤니티 참여가 프로젝트를 10배 성장시킴

**④ 실제 생물학 데이터 기반**
- 상상이 아니라 실제 커넥톰, 실제 전기생리학 데이터 사용
- C. elegans 302뉴런 커넥톰은 공개 데이터

### 실패 패턴 (반드시 피할 것)

**① 과도한 약속 (Blue Brain의 실수)**
- Henry Markram: "10년 안에 인간 뇌 시뮬레이션" → $10억+ 투입 후 좌초
- 원인: 데이터 수집 ≠ 이해. 모은다고 사고가 나오는 게 아님

**② 지나친 상세함 (속도 함정)**
- Hodgkin-Huxley 모델: 정확하지만 느림. 대규모 불가
- LIF 모델: 덜 정확하지만 초파리 전체 뇌 가능 → 이쪽이 이김

**③ 커넥톰만 의존 (구조 함정)**
- 배선도만으로는 행동 예측 불가
- 신경조절(도파민, 세로토닌)이 빠지면 "감정 없는 로봇"

**④ 혼자만의 프로젝트**
- 공개하지 않으면 검증도 못 받고 성장도 못 함

## 3. 우리에게 가장 가까운 레퍼런스

### 1순위: Numenta (Jeff Hawkins)

**왜:** 우리와 철학이 거의 같다.
- "뉴런 메커니즘의 본질을 이해해야 진짜 지능을 만들 수 있다"
- 피질 컬럼 = 반복되는 기본 단위 (우리의 "뉴런 → 연결 → 창발" 철학과 동일)
- Thousand Brains Theory = ASTRAL NOVA의 "아니마가 각자 세계를 관측"과 구조적으로 동형

**배울 점:**
- Monty 프레임워크 코드를 참고할 수 있음 (오픈소스)
- 감각-운동 통합이 핵심이라는 것
- 피질 컬럼의 구체적 구현 방법

### 2순위: FlyWire + OpenWorm

**왜:** "실제 뇌를 그대로 옮겨서 돌리면 그 생물처럼 행동한다"는 증명.
- 베스퍼님이 영감을 받은 바로 그 접근법
- LIF 모델 + 실제 커넥톰 = 노트북에서 작동

**배울 점:**
- 우리도 C. elegans 302뉴런부터 시작할 수 있음 (커넥톰 데이터 공개)
- LIF 모델이면 충분하다는 실증
- Brian2 시뮬레이터 사용법

### 3순위: Nengo/Spaun (Chris Eliasmith)

**왜:** "기능하는 뇌 모델"의 유일한 실증.
- 250만 스파이킹 뉴런으로 8가지 인지 작업 수행
- 암기, 추론, 의사결정, 문제 해결 등

**배울 점:**
- 뉴런에서 인지까지 가는 구체적 경로
- Neural Engineering Framework (뉴런→기능 변환 수학)

## 4. 모든 프로젝트에서 공통으로 중요한 메커니즘

4팀의 조사를 종합하면, 성공한 프로젝트들이 공통으로 구현한 핵심 메커니즘:

| # | 메커니즘 | 모든 프로젝트에서? | 설명 |
|---|---------|:---:|------|
| 1 | **스파이크 타이밍** | ✅ | 언제 발화하는가가 핵심 (빈도가 아님) |
| 2 | **이벤트 구동** | ✅ | 필요할 때만 계산 (항상 돌리지 않음) |
| 3 | **STDP (학습)** | ✅ | 타이밍 기반 시냅스 강화/약화 |
| 4 | **억제/흥분 균형** | ✅ | 억제 뉴런이 있어야 안정적 |
| 5 | **정규화** | ✅ | 상대적 신호 크기로 비교 (절대값 아님) |
| 6 | **피드백 루프** | ✅ | 상위→하위 역방향 연결 |

추가로 중요하지만 아직 실험적인 것:
- 신경조절 (도파민/세로토닌 등 글로벌 모드 전환)
- 확률성 (시냅스 방출 확률)
- 전기적 시냅스 (gap junction)

## 5. 우리의 다음 한 걸음 — 제안

조사 결과를 종합하면, 가장 현실적이고 검증된 경로:

### Phase 0: 도구 확보 (즉시)
- Brian2 설치 + 기본 LIF 뉴런 시뮬레이션 실행
- C. elegans 커넥톰 데이터 다운로드 (OpenWorm 공개 데이터)

### Phase 1: C. elegans 부분 회로 (1-2개월)
- 302뉴런 중 20-30개짜리 회로 선택 (먹이 회피 등)
- LIF + 실제 커넥톰 연결 → 시뮬레이션
- 실제 선충 행동과 비교 검증

### Phase 2: 학습 추가 (2-4개월)
- STDP 적용 → 경험으로 행동 변화하는지 확인
- 우리의 "세 패턴 + 확률성 + 동기화" 가설 검증

### Phase 3: 확장 (4-6개월)
- 초파리 수준으로 규모 확대
- 또는 피질 컬럼 1개 구현 (Numenta 참고)

→ **베스퍼님 판단 필요**: 이 경로로 갈지, 다른 방향을 원하시는지.
