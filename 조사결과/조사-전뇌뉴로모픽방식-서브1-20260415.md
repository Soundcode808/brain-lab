# 서브1 조사결과 — 전뇌/뉴로모픽 방식 탐색

> 조사일: 2026-04-15 | 담당: 서브1

## 전뇌 에뮬레이션 / 뉴로모픽 현황

### Whole Brain Emulation (WBE)
- State of Brain Emulation Report 2025 발표 (175p, 45명+ 전문가)
- 현재: C. elegans(302 뉴런) → 과일파리(10만 뉴런) 매핑 완성
- 미달성: 95%+ 신경세포 단일 스파이크 해상도 동시 기록 아직 없음
- 전 세계 WBE 종사자 500명 미만

### 뉴로모픽 컴퓨팅 — 실용 단계
- Intel Loihi 2, IBM NorthPole: GPU 대비 1000배 에너지 효율
- 적용: 자율주행, 사이버보안, 헬스케어, 로봇

### Spiking Neural Networks (SNN)
- 생물학적 뉴런 모방, 이산 스파이크 정보 전달
- BrainCog: 지식 그래프를 SNN으로 표현
- Loihi 2 + IBM NorthPole + 도메인 LLM: hallucination 68% 감소

## 위키/MD 한계를 넘는 새로운 접근

### Memory-Augmented Neural Networks (MANN)
- 외부 메모리 모듈 + 뉴럴 컨트롤러
- 텍스트 검색보다 빠름, 구조적 관계 자동 학습
- 현재 RAG, 대화형 AI에 적용 중

### Neural Turing Machine (NTM) & DNC
- DeepMind 2014~2016. 외부 메모리 읽/쓰 가능
- 패턴 매칭(뉴럴) + 알고리즘(튜링) 결합
- 2025: Neural Field Turing Machine → PDE 물리, 이미지 복구

### Graph Neural Networks + Hierarchical Memory
- GrapHD: 뇌 같은 인지 학습
- 텍스트 파일 = 정적 vs 그래프 메모리 = 동적 구조 학습

### HTM (Hierarchical Temporal Memory) — Numenta
- Jeff Hawkins "On Intelligence" 기반
- Accelerated HTM: 반복 처리 7.55배 가속
- 이상 탐지(NAB) 최고 성능

## 비슷한 방향의 사람들/프로젝트

### OpenCog Hyperon
- 분산 하이퍼그래프(Atomspace) + 자기수정 언어(MeTTa)
- 목표: 에머전트 AGI. 2025 알파 버전
- SingularityNET 통합 중

### BrainCog
- 50+ SNN 알고리즘. 초파리~인간 뇌 다중 스케일
- "뇌 메커니즘 해독 + 인공 뇌 개발" 명시 목표

### Neural Brain (2025)
- 뇌과학 기반 embodied AI 통일 프레임워크
- 다중모달 센싱 + 인지-행동 + 신경가소성 메모리
- "Mental World Model"로 인간 의도 이해

### Agentic AI 수렴 트렌드
- 모놀리식 LLM → 모델 함대(전문가 소규모 모델들)
- 100+ 인지 아키텍처 → 공통 기능 구조 합의 형성 중

## 지금 쓸 수 있는 것 vs 미래

### 지금 가능 (2025-2026)
- 뉴로모픽 하드웨어 (Loihi 2, NorthPole) — 상용화
- MANN — 엔터프라이즈 RAG, 챗봇
- SNN — BrainCog, 이상 탐지
- HTM — 시계열 이상 탐지
- GNN + Hierarchical Memory — 지식 표현
- OpenCog Hyperon — 프로토타입

### 근시일 (2026-2028)
- Sub-million neuron WBE (초파리/제브라피시)
- Meta V-JEPA 2 (영상 학습 → 로봇)
- Continual Learning in Agents

### 먼 미래 (2030+)
- 전체 인간 뇌 에뮬레이션
- 뉴로모픽 AGI
- 완전 자율 뇌 기반 아바타

## 핵심 인사이트
1. 뉴로모픽/뇌 영감 기술은 상용화 단계 진입 (Loihi 2, NorthPole)

3. 메타버스 + 뉴로모픽 AI 결합 이미 연구 중 (Neural Brain, V-JEPA 2)
