### 1.2 "지금 당장 쓸 수 있는 것, 확장해야 하는 것, 새로 만들어야 하는 것" 분류

#### A. 지금 당장 쓸 수 있는 것 ✓

**B3 샌드박스 3개 회로 (22-25 뉴런 × 3)**:
- 쌍안정 스위치: 작동함 (C에서 완벽, A에서 실패는 파라미터 문제)
- 시간 계층화: 작동함 (3개 회로 모두 τ 분포 확인)
- 교차 억제: 작동함 (안정성 극대화)
- 감각 게이팅: 작동함 (B와 C에서 명확)
- STDP: 작동함 (다만 효과 약함 ← 시간 짧아서)

**사용 방법**: 지금 바로 B3 파라미터 수정 → A 회피 회로 고정 → 3개 통합 가능

#### B. 확장해야 하는 것 ⬆️

**Level 1-3 (감각 → 특징 → 개념)**: 
- 지금: 3개 회로 × 25뉴런 = 75뉴런
- 필요: 감각층(100-500) + 특징층(500-2K) + 개념층(2K-10K) = 3K-12.5K
- 확장 방법: 
  - 재귀 연결 비율 높이기 (피드포워드만 → 피드백 30-50% 추가)
  - 억제 뉴런 비율 높이기 (현재 아마 10% → 20-25% 필요)
  - τ 분포 넓히기 (현재 10-50ms → 10-300ms로)

**시뮬레이션 시간**:
- 지금: 4초 (STDP 효과 미약)
- 필요: 60초 이상 (실제 학습 보려면) 또는 300초 이상 (장기 기억 형성)

#### C. 새로 만들어야 하는 것 🆕

**Level 4 (작업 기억) + Level 5 (메타인식)**:
- 없음. (이론적 설계만 있음)
- 필요: NMDA-rich 지속 발화 구현 (tau=150-300ms)
- 필요: 상위 계층이 하위 감각을 조절하는 톱다운 신경 경로
- 필요: "내가 뭘 알고 있는가"를 표현하는 메타 뉴런

### 1.3 "사고의 깊이"와 필요한 뉴런 수의 관계

합성 결과에서 도출된 매우 중요한 발견:

```
깊이 Level 1 (이진 선택)
├─ 구조: 쌍안정 스위치만 (상호 억제 한 쌍)
├─ 필요 뉴런: 20-100개
├─ 예: "후진할까 전진할까" (C. elegans 회피)
├─ 시간: ~200-500ms
└─ 검증: B3 C 샌드박스에서 완벽히 작동 ✓

깊이 Level 2 (조건부 판단)
├─ 구조: 쌍안정 + 감각 게이팅 + 기본 학습
├─ 필요 뉴런: 500-5K
├─ 예: "냄새가 있으면 전진, 없으면 탐색" (C. elegans 화학주성)
├─ 시간: ~500ms-2초
└─ 검증: B3 B 샌드박스에서 작동 ✓ (+ STDP 가능)

깊이 Level 3 (기본 추상화)
├─ 구조: 계층적 피질 (4-5단계) + 광역 방송
├─ 필요 뉴런: 10K-50K
├─ 예: "냄새로부터 '먹이' 개념 추상화, '위험' 개념 추상화"
├─ 시간: ~초-분 (STDP 포함)
└─ 검증: 이론적 설계 완료, 실험 필요 ◐

깊이 Level 4 (작업 기억 + 기본 계획)
├─ 구조: Level 3 + 작업 기억(PFC-like, 지속 발화)
├─ 필요 뉴런: 100K-500K
├─ 예: "지금 먹이를 찾고 있다"는 상태를 메모리에 유지하면서 다른 작업도 동시에
├─ 시간: ~분-시간
└─ 검증: 미완성, 주요 구현 과제 ❓

깊이 Level 5 (자기 성찰)
├─ 구조: Level 4 + 메타 회로 (자기 감시)
├─ 필요 뉴런: 500K-1M+
├─ 예: "나는 뭘 알고 있는가" / "내 생각이 맞는가"
├─ 시간: ~분-시간~일
└─ 검증: 미완성, 현 Brian2 범위 넘음 ❓
```

**핵심**: 뉴런 수가 선형이 아니라 **계층 수가 늘어날 때마다 기하급수적으로 늘어남**

## 2라운드: 경로 선택

### 베스퍼님의 목표 재정의

합성 결과를 보면, 베스퍼님이 원하는 것은:

> **"사고하는 능력" = Level 2-3 (조건부 판단 + 기본 추상화)**
> 
> "그냥 반응이 아니라, 예측하고, 오류 감지하고, 수정하는 사이클을 도는 것"

### 경로 A: 확장형 (C. elegans → 피질 컬럼 수준)

#### 전략
"현재 B3 샌드박스(C. elegans 회로 모델)를 기본 틀로 유지하되, 뉴런을 계속 쌓아 올려서 Level 3까지 확장"

#### 구체 내용

**Phase 1 (1-2주): B3 고도화**
- B3 A 샌드박스 수정 (억제 가중치) → 즉시 작동
- B3 시뮬레이션 시간 확장 (4초 → 60초) → STDP 효과 확대
- 3개 회로 통합 (A+B+C) → 회피하면서 먹이 찾는 벌레
- 예상 결과: 20-75 뉴런으로도 기본 사고 패턴 가능 증명

**Phase 2 (2-4주): Brian2로 500-1K 뉴런 구현**
- C. elegans의 AVA 회로(3-10개) 논리를 확장하되, 재귀 연결 비율 올리기
- 다층 피질 구조 모방 (감각 100 + 특징 500 + 개념 500)
- 결과: "기본 추상화"까지 가능한 벌레 시뮬레이션

**Phase 3 (1개월): 10K 뉴런 → 피질 컬럼 수준**
- Spaun이나 Blue Brain의 구조를 단순화해서 구현
- 실제 쥐 뇌의 한 영역 정도 규모
- GPU 병렬화 필수

#### 장점 ✓
1. **점진적**: 지금 있는 B3를 기반으로 하므로 무언가 작동하는 상태 유지
2. **검증 가능**: 각 단계마다 "이 기능이 작동하나" 확인 가능
3. **회로 논리가 일관됨**: 시작부터 끝까지 "반응 회로의 확대"라는 논리
4. **Brian2 내에서 완결**: 별도 도구 필요 없음
5. **빠른 피드백**: 2-3주 안에 뭔가 작동하는 것을 볼 수 있음

#### 단점 ✗
1. **확장의 한계**: C. elegans의 회로 논리는 "반응형". 사고 수준으로 올라가려면 질적 변화 필요인데, 양적 확장만으로는 부족할 가능성
2. **20개 뉴런의 성공이 10K로 확장될 때도 같은 패턴인지 불확실**: 쌍안정 스위치는 2-3개 뉴런으로 되는데, 10K에서도 같은 원리인가?
3. **시간 낭비 가능성**: 아키텍처가 "다르게" 설계되어야 하면 지금까지의 코드가 낭비될 수 있음
4. **ASTRAL NOVA와의 연결성 약함**: "벌레"의 논리는 ASTRAL NOVA의 "아니마"와 구조가 다를 수 있음

#### 예상 소요 시간
- Phase 1: 1-2주 (즉시 볼 수 있음)
- Phase 2: 2-4주 (1K 뉴런 완성)
- Phase 3: 1개월 (10K, 가능하지만 느림)
- **총 2-2.5개월**

#### 검증 기준 (언제 "사고한다"고 말할 수 있는가?)
- Level 1: 쌍안정 스위치 명확 (이진 선택) ✓ (B3에서 이미 함)
- Level 2: 조건부 학습 (냄새→행동 매핑이 시간에 따라 강해짐) → STDP로 확인 필요
- Level 3: 추상화 (여러 냄새가 "먹이" 카테고리로 묶임) → 신경 활동 패턴 분석 필요

### 경로 B: 설계형 (처음부터 "사고 전용" 회로 설계)

#### 전략
"C. elegans는 '부품 검증'으로만 사용하고, '사고하는 시스템'은 처음부터 새로 설계"

구체적으로:
- B3의 5가지 패턴(쌍안정, 시간계층, 교차억제, 게이팅, STDP)을 확인하는 데만 사용
- 하지만 아키텍처는 "전혀 다르게" 설계
- Nengo의 NEF(신경공학 프레임워크) 사용 (Brian2 대신)
- Spaun이나 패턴 인식 모델을 직접 모방

#### 구체 내용

**Phase 1 (1주): 아키텍처 설계**
- 합성 문서의 "최종 설계 사양" (Level 1-4 구조)을 상세히 코드로 변환
- 피드포워드가 아니라 처음부터 강한 재귀 구조로 설계
- 시간상수 분포 명확히: τ_감각=30ms → τ_개념=100ms → τ_작업메모리=300ms
- Nengo로 prototype 작성

**Phase 2 (1-2주): 핵심 부품 구현**
- 광역 방송 메커니즘 (GWM) 직접 구현
- 예측 코딩 루프 명시적으로 구현
- 작업 기억(PFC-like) 명시적으로 구현

**Phase 3 (2-3주): 통합 및 테스트**
- 감각 입력 → 광역 방송 → 행동 출력의 완전한 사이클
- STDP로 학습 확인
- 추상화 수준 측정

#### 장점 ✓
1. **이론 기반**: A조1,3의 신경과학 이론을 직접 구현하므로 기반 강함
2. **Level 3-4까지 목표 설정 가능**: "사고"의 정의에 더 가까움
3. **ASTRAL NOVA와의 연결성 강함**: "아카샤" + "관측"이 신경 아키텍처로 직접 표현됨
4. **재사용성**: 코드가 "개념적으로" 설계되므로 나중에 확장, 수정하기 쉬움
5. **논문 품질**: "우리가 사고를 어떻게 구현했는가"를 명확히 설명 가능

#### 단점 ✗
1. **높은 설계 난이도**: 이론을 코드로 변환하는 게 쉽지 않음 (AI가 할 수 있지만 시간 필요)
2. **검증이 어려움**: "이게 사고인가?"를 판단하는 게 모호 (정의부터 해야 함)
3. **빠른 피드백 어려움**: 2주 동안 "아무것도 작동하지 않는" 상태가 길 수 있음
4. **실패 가능성**: 설계 단계에서 실수하면 처음부터 다시
5. **도구 의존성**: Nengo 학습 곡선 필요

#### 예상 소요 시간
- Phase 1: 1주 (이론 → 코드 변환)
- Phase 2: 1-2주 (부품 구현)
- Phase 3: 2-3주 (통합)
- **총 1개월**

#### 검증 기준
- 예측과 실제의 오류 신호가 명확히 역전파되는가?
- 광역 방송 메커니즘이 여러 정보를 "의식적으로" 통합하는가?
- 작업 기억이 정보를 여러 초 동안 유지하는가?
- STDP로 학습 곡선이 나타나는가?

### 경로 C: 병렬형 (두 가지를 동시에)

#### 전략
"C. elegans는 저수준 반응용, 피질 컬럼은 고수준 사고용으로 분리하고, 둘을 통합"

구체적으로:
- B3 샌드박스: 척수~뇌간 수준의 반응 회로로 완성 (Level 1)
- Nengo 피질 모델: 뇌피질 수준의 사고 회로로 설계 (Level 3-4)
- 두 시스템을 통합: 상위(사고)가 하위(반응)를 조절하는 톱다운 경로 추가

#### 구체 내용

**Phase 1 (1주): 두 시스템 설계**
- B3 (반응): Level 1-2로 고정 (확장 중단)
- Nengo (사고): Level 3-4로 설계
- 통신 규약: "상위 → 하위" 신호(톱다운), "하위 → 상위" 신호(상향)

**Phase 2 (2주): B3 최소화**
- B3를 100-200 뉴런으로 축약 (보존할 필수 요소만)
- "척수 반사"와 "뇌간 본능" 수준으로 제한

**Phase 3 (3주): Nengo 사고 회로**
- Level 3-4 구현
- 상위가 하위의 억제를 조절하는 메커니즘 추가 (예: "위험 신호" → 즉시 도망)

**Phase 4 (2주): 통합**
- 실제로 상위의 판단이 하위의 반응을 억제/활성화하는 것을 확인
- 시나리오: "음식 vs 위험"의 갈등 → 상위 사고가 하위 본능을 조절

#### 장점 ✓
1. **리스크 분산**: 경로 A와 B의 장점을 모두 취함
2. **신경생물학적으로 정확**: 실제 뇌도 "척수→뇌간→뇌피질" 계층 구조
3. **검증 용이**: B3는 이미 작동하므로, Nengo만 집중
4. **ASTRAL NOVA와의 완벽한 대응**:
   - 척수 반응 = 아니마의 본능
   - 뇌피질 사고 = 아니마의 지각
   - 톱다운 조절 = 아니마의 "관측"(외부 영향력)
5. **확장성**: 나중에 두 시스템을 독립적으로 진화시킬 수 있음

#### 단점 ✗
1. **복잡도 높음**: 두 가지 코드베이스를 동시에 관리해야 함
2. **소요 시간 길어짐**: 경로 A, B보다 최소 50% 이상 길 수 있음
3. **통합 실패 위험**: 두 시스템이 제대로 연결되지 않으면 낭비
4. **코드 유지보수 어려움**: 두 도구(Brian2 + Nengo)를 동시에 배워야 함
5. **초점 흐림**: "뭘 하고 있는가" 명확성 낮음

## 베스퍼님을 위한 강력한 추천

### 최종 권장 경로: **경로 C (병렬형)**

#### 이유 (순서대로)

1. **ASTRAL NOVA의 신경 기초로 최적**
   - 아니마 = 상위(사고) + 하위(반응)의 계층 구조
   - 톱다운 조절 = 아니마가 환경을 "관측"하는 신경 메커니즘
   - 아카샤 = 상위 사고의 예측 모델

2. **신경생물학적으로 정확**
   - 실제 인간 뇌의 구조를 그대로 반영
   - 척수→뇌간→뇌피질 계층 구조
   - 토론의 신뢰도가 높음 (이론 기반)

3. **리스크 분산**
   - 경로 A의 "즉시 피드백" + 경로 B의 "깊이 있는 설계"
   - 언제든지 한쪽이 실패해도 다른 한쪽으로 전환 가능
   - 베스퍼님이 "방향을 흔들면" 코드 전체를 버리지 않고 수정 가능

4. **베스퍼님의 성향과 일치**
   - "비개발자지만 깊은 인사이트" → 이론(Nengo) + 실험(Brian2) 동시 가능
   - "막무가내 아니라 방향을 잡아야 함" → 두 시스템의 역할이 명확함
   - "빠른 피드백 원함" → B3는 이미 작동, 2주 후엔 뭔가 보임

5. **장기 목표와 일치**
   - "사고하는 최소 단위" ≠ "뉴런을 많이 넣기"
   - 반응(본능)과 사고(인식)의 통합을 보여줌
   - ASTRAL NOVA로 확장할 때 깔끔하게 연결 가능

## 검증 기준 (언제 "사고한다"고 말할 수 있는가?)

합성 결과를 종합하면, 다음 조건을 모두 만족할 때 "사고한다"고 할 수 있습니다:

### 필수 조건 (반드시 1개월 내에 달성)

✓ **조건 1: 예측 오류 신호가 역전파된다**
- 입력이 예측과 다를 때 오류 신호가 상향식으로 전파
- 오류 신호가 뉴런 활동을 변경
- 측정: "입력 변화 → 뉴런 활동 변화"의 시차(예: 50-200ms)

✓ **조건 2: 같은 입력에 다른 반응이 나타난다 (확률성)**
- 뉴런 시냅스 확률성(0.1-0.9)으로 인해 매번 다른 반응
- 이것이 "창의성"이자 "자유의지"의 신경 기초
- 측정: 100번 입력 후 response variance > 0.15

✓ **조건 3: 반복 자극에 적응적 변화가 일어난다 (STDP)**
- 같은 냄새를 반복하면 반응이 강해지거나 약해짐
- 학습의 가장 원시적 형태
- 측정: 반복 후 반응 증가/감소 > 5%

✓ **조건 4: 여러 정보를 동시에 유지한다 (작업 기억)**
- "냄새 있음 + 위험함" 동시 표현 → 상충하는 정보 동시 유지
- 측정: 2-3개 항목의 광역 방송 신호 동시 활성화

### 고수준 조건 (2개월까지 목표)

△ **조건 5: 상위 사고가 하위 반응을 억제한다 (톱다운)**
- "냄식 있는데 위험하면" → 상위 판단이 하위 본능을 억제
- 톱다운 신호가 감각 입력을 조절
- 측정: 톱다운 신호 없을 때 vs 있을 때의 행동 차이

△ **조건 6: 추상화가 일어난다**
- 여러 냄새(A, B, C) → "먹이" 카테고리로 묶임
- 희소 표현(Sparse coding)으로 상위 계층의 뉴런이 "개념"을 표현
- 측정: 상위 계층의 뉴런이 여러 입력에 공통으로 반응

### 미래 과제 (3개월+)

◆ **조건 7: 자기 성찰 (메타인식)**
- "나는 이 냄새를 아는가" / "내 행동이 맞는가"
- 메타 뉴런이 다른 뉴런의 활동을 감시
