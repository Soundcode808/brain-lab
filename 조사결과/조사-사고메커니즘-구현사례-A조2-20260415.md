## 1. Spaun / Nengo: 대규모 스파이킹 뉴런 모델

### 1.1 Spaun 기본 구조
- **뉴런 규모**: 2.5백만 개의 스파이킹 뉴런
- **뇌 영역 시뮬레이션**: 20개 뇌 영역 통합
- **수행 능력**: 8가지 인지 작업
  - 숫자 인식 (Visual pattern recognition)
  - 리스트 암기 (List memory)
  - 덧셈 (Addition)
  - 패턴 완성 (Pattern induction/completion)
  - 강화학습 기반 의사결정
- **입출력**: 28×28 망막 시뮬레이션 → 6근육 팔 제어

### 1.2 추론 회로의 구현 메커니즘

#### NEF의 세 가지 핵심 원리 (Neural Engineering Framework)

**원리 1: 표현 (Representation)**
- 뉴런 집단이 실수 벡터를 비선형 인코딩/선형 디코딩으로 동시 표현
- LIF(Leaky Integrate-and-Fire) 뉴런 모델 사용
- 각 뉴런의 인코더: 뉴런이 반응하는 신호 방향 정의
- 각 뉴런의 디코더: 스파이크 열차에서 신호 추출

**원리 2: 변환 (Transformation)**
- 함수 계산은 다음 공식으로 연결가중치 결정:
  ```
  Connection Weights = Decoder(함수) × Encoder(타겟 집단)
  ```
- 선형 및 비선형 변환 모두 가능
- 뉴런 간 연결을 해석적으로 계산 (역제곱법)

**원리 3: 동역학 (Dynamics)**
- 표현값을 동역학계의 상태변수로 간주
- 재귀 연결을 변환 원리로 계산
- 제어 이론 및 동역학계 이론 직접 적용 가능
- 통합기(integrator), 발진기, 칼만필터 등 구현 가능

#### Spaun의 추론 회로 구조

**패턴 유도(Inductive Reasoning) 구현**:
1. **시각 입력층**: 28×28 망막 → 시각 특징 추출
2. **기억 버퍼**: 표현 원리로 구현된 작업메모리
   - 재귀 연결로 정보 유지
   - 지속 활성(persistent firing)은 강한 재귀 양성피드백으로 구현
3. **규칙 학습층**: 패턴 관계 학습
   - 입력 패턴들의 공통 구조 검출
   - 뉴런 집단의 분산된 표현으로 규칙 부호화
4. **응답 생성층**: 학습된 규칙으로 다음 항 예측
5. **모터 제어**: 팔 움직임으로 답 표현

**키 메커니즘**:
- 신경 인코더의 중복성(redundancy)이 노이즈 허용
- 저랭크 분해(low-rank factorization) 덕분에 뉴런 수 실용적
- 스파이크 타이밍의 정확성보다 집단 코딩이 핵심

### 1.3 Nengo 구현 도구

- Python 기반 신경망 컴파일러
- NEF 원리를 고수준 지시에서 저수준 뉴런 모델로 자동 변환
- GPU 가속 가능 (GeNN, CUDA 확장)
- 신경보학 하드웨어(neuromorphic hardware)로 직접 컴파일 가능

## 2. 피질 컬럼 시뮬레이션

### 2.1 Blue Brain Project 사례

#### 기본 성과
- **첫 성공**: 2015년 쥐 신피질 컬럼의 30,000 뉴런 시뮬레이션
- **9년 투자 후**: 상세하고 검증된 정량적 모델 완성
- **프로젝트 종료**: 2024년 12월 (독립 재단으로 전환)

#### 발견사항

**신경-아교세포 상호작용**:
- 신경-아교세포-혈관 단위(neuro-glial vascular unit) 역할 규명
- 뇌 에너지 관리 메커니즘 모델화

**신경집합의 다차원 구조**:
- 신경망이 정보 처리 시 형성하는 고차원 구조 발견
- 신경 연결성과 세포 집합 형태의 관계 규명

**신경 형태 합성 (2022)**:
- 위상대수학(algebraic topology) 기반 알고리즘 개발
- 적은 샘플로 수백만 개의 고유한 뉴런 형태 합성
- 건강한 상태와 질병 상태 모두 재현 가능

### 2.2 Numenta Monty 프레임워크

#### 아키텍처 원리

**기본 단위: 학습 모듈 (피질 컬럼 모델)**
- 계층적 시간 기억(HTM) 신경세포로 구성
- 각 세포의 3가지 상태: 활성(active), 예측적(predictive), 비활성(inactive)

**위치층**:
- 격자세포(grid cell) 모듈
- 삼각형 격자 배열의 세포들
- 감각 입력의 공간적 맥락 부호화

**감각층**:
- 미니컬럼(mini-column) 구조
- 선형 배열된 세포들
- 국소 특징 인식

**측면 연결과 협력**:
- 컬럼 간 부분 정보 공유
- 대상 인식 속도 증가
- 계층적 및 비계층적 상호작용 모두 지원
- 신경 메시징 프로토콜(CMP)로 다중감각 통합

#### 피질 컬럼의 한계와 가능성

**가능한 것**:
- 단일 객체의 완전한 모델링
- 공간 참조틀 기반 표현
- 운동 추정 및 제어

**제한**:
- 단일 컬럼으로는 추상적 추론 불가
- 계층적 통합 필요
- 뇌 전역의 신경조절 신호 포함 필요

## 3. 작업메모리 회로 구현

### 3.1 지속 발화 메커니즘 (Persistent Firing)

#### 기본 원리

**재귀 흥분 아키텍처**:
- 강한 재귀 양성피드백이 핵심
- 외부 자극 제거 후에도 신경 활동이 자기유지됨
- 선택적 지연 활성(selective delay activity)은 개별 뉴런의 성질이 아님
- **집단 역학(population dynamics)**: 상호 흥분하는 뉴런 부분집합의 집단적 특성

#### 단기 시냅스 가소성의 역할

**촉진(Facilitation)**:
- 단기 촉진으로 시냅스 강도 증가
- 기억 보유 안정화
- 확산(diffusion) 감소, 방향 표류 감소

**억압(Depression)**:
- 단기 억압으로 시냅스 강도 감소
- 기억 변동성 증가
- 회로 적응성 향상

#### 좁고 넓은 스파이킹 뉴런의 역할

**좁은 스파이킹 뉴런(Narrow-spiking)**: 
- 국소 흥분성 및 억제성 신경세포
- 지속 활성 생성의 직접 기여
- 기억 정확도의 핵심 예측인자

**넓은 스파이킹 뉴런(Broad-spiking)**:
- 피라미드형 투사 뉴런
- 뇌 영역 간 신호 전달
- 기억 표현의 개방성 제공

#### 전전두엽 피질(PFC)의 신경조절

**아세틸콜린(Acetylcholine)**:
- PFC 어트랙터 역학 조절
- 작업메모리 성능 제어
- 신경 흥분성과 기억 정밀도 균형

### 3.2 어트랙터 네트워크 모델

#### 연속 어트랙터 역학

**기본 구조**:
```
x(t+1) = f(Wx(t) + u(t))
```
- x: 신경 활동 상태
- W: 재귀 가중치 행렬
- u: 입력 신호
- f: 활성화 함수

**안정성 조건**:
- 재귀 연결의 강도에 따라 특정 상태로 수렴
- 범프 어트랙터(bump attractor): 공간 작업메모리의 핵심
  - 신경 활동의 공간적 범프가 기억 내용 부호화
  - 시각적 주의, 안구운동 계획, 팔 움직임 등에 사용

**신경생물학적 구현**:
- 쥐 전전두엽: 지연 반응 작업에서 범프 어트랙터 확인
- 재귀 연결의 강한 흥분성 상호작용
- 짧은 잠복 시간의 시냅스 전달

#### 장기 안정성: 활동-침묵 상호작용

**최근 발견 (2024)**:
- 지속 활성(persistent activity) + 활동-침묵 동역학(activity-silent dynamics)의 상호작용
- 침묵 상태에서의 신경 재활성화가 연쇄 편향(serial bias) 생성
- 이중 메커니즘: 지속 활성으로 주요 정보 유지, 침묵 동역학으로 간접 정보 유지

## 4. 최소 사고 회로의 규모와 조건

### 4.1 생물학적 최소 시스템

#### C. elegans: 302 뉴런의 의사결정

**전체 신경계**:
- 302개 뉴런
- 약 7,000개 시냅스
- 완전 연결성 지도 존재

**의사결정 회로의 핵심**:
- **AVA 뉴런**: 허브 역할 (중심 신경절의 premotor interneuron)
  - 위협(tail) 입력: 억제성(GABA) 신호
  - 보상(head) 입력: 흥분성(glutamate) 신호
  - 공간 대칭성: 억제성 수용체와 흥분성 수용체가 다른 위치 분포
- **AVB 뉴런**: 전진 운동 제어
- **PVP 뉴런**: 조절(modulatory) - AVB 활성화, AVA 억제
- **DVC 뉴런**: 이제억제(disinhibition) 회로 - PVP 억제로 AVA 활성화

**회로 메커니즘**:
- 신경전달물질의 공간적 분포 차이로 신호 분리
- 각 결정(전진 vs 후진)에 5-10개 뉴런의 작은 부분집합 관여
- 단순한 2-3단계 회로로 복잡한 선택지 통합

#### Drosophila 버섯체(Mushroom Body): 약 2,200 뉴런

**구조**:
- Kenyon 세포(KC): 약 2,000개의 고도로 희소한 표현
- 출력 뉴런(MBON): 34개 (21가지 유형)
- 도파민 뉴런(DAN): 20가지 유형

**학습 메커니즘**:
- 감각 입력: 2,000개 KC의 희소 표현으로 변환
- 가치 신호: 20가지 DAN이 15개 구획에 평행 배열에 투사
- 학습: DAN의 강화 신호가 KC-MBON 시냅스 강도 변경
- 행동: MBON 활성화가 모터 신호로 직접 변환

**최소 성숙 시스템**:
- 유충 초기 단계: 약 44개 뉴런으로 학습 가능
- 성숙 단계: 약 150-200개 뉴런의 확장된 회로

### 4.2 인공 신경망에서의 최소 회로

#### Nature 2025 발견: 극도로 미니 신경망

**놀라운 결과**:
- 1-4개 유닛의 신경망이 동물의 행동 선택 예측 가능
- 고전 인지 모델들을 능가
- 6가지 보상 학습 과제에서 큰 신경망과 동등 성능

**최적 크기 이론**:
- 생물학적으로 현실적인 시냅스 잡음이 있을 때, 주어진 작업에 최적 네트워크 크기 존재
- 최적 크기 이상으로 뉴런 증가 시 학습 성능 저하
- 신경세포는 복잡한 행동 **또는** 빠르고 정확한 학습 중 선택
- 더 많은 뉴런 = 더 복잡한 제어 권한 아님

**임계값이 아닌 최적점**:
- 의사결정에 필요한 뉴런: 작업 유형에 따라 3-20개 범위
- 회로 아키텍처가 뉴런 수보다 중요
- 특정 상태 제어 부분집합: 전체 회로보다 훨씬 작음

### 4.3 "사고"의 정의와 회로 규모

#### 최소 사고 조건

**3가지 필수 요소**:
1. **정보 표현**: 현재 상태, 기억, 예측 상태 부호화
2. **변환**: 표현된 정보에 함수 적용 (논리 연산, 패턴 매칭 등)
3. **재귀 동역학**: 자기 피드백으로 시간 경과에 따른 상태 유지

#### 구현 최소 단위

| 시스템 | 뉴런 수 | 주요 기능 | 복잡도 |
|--------|--------|---------|--------|
| C. elegans AVA 회로 | 3-10 | 이진 의사결정 (전진/후진) | 낮음 |
| 보상 학습 미니망 | 1-4 | 단순 강화학습 | 낮음 |
| Drosophila 유충 | 44 | 사탕맛/악취 학습+기억 | 중간 |
| 공간 작업메모리 범프 | 50-100 | 위치 기억 (초 단위) | 중간 |
| Spaun 시각 추론 부분 | 수만 | 패턴 유도, 숫자 인식 | 높음 |
| 전체 Spaun | 250만 | 8가지 인지 작업 동시 수행 | 매우 높음 |

## 5. Brian2로 구현 가능성 평가

### 5.1 Brian2의 강점

✓ **NEF 호환성**: Nengo 모델을 Brian2로 변환 가능  
✓ **LIF 뉴런**: Spaun과 동일한 모델 지원  
✓ **재귀 네트워크**: 작업메모리 어트랙터 구현 가능  
✓ **스파이킹 시뮬레이션**: 정확한 시간 역학 제공  
✓ **GPU 가속**: 중규모 네트워크 실시간 실행 가능  
✓ **유연성**: 맞춤형 신경 방정식 정의 가능  

### 5.2 실현 가능한 프로젝트 범위

#### 즉시 구현 가능 (1-3개월)
- ✓ C. elegans AVA 의사결정 회로 (5-10개 뉴런 완전 모델)
- ✓ 단순 지속 활성 메모리 (50-100개 뉴런의 어트랙터)
- ✓ 보상 학습 회로 (수백 뉴런, Drosophila 유충 수준)
- ✓ NEF 변환 원리 시연용 소형 모델 (500-1000 뉴런)

#### 중기 프로젝트 (3-12개월)
- ◐ 패턴 유도 회로 (수천 뉴런, Spaun 시각 부분의 단순화)
- ◐ 계층적 메모리 시스템 (수만 뉴런)
- ◐ 다중 어트랙터 시스템 (경합하는 결정 상태)

#### 도전적 프로젝트 (12개월 이상)
- ◆ 전체 Spaun 복제 (250만 뉴런)
  - GPU 클러스터 필요
  - 시뮬레이션 시간: 실시간의 10-100배
  - Nengo와 협력 권장

### 5.3 구현 시 주의사항

**재귀 연결 설계**:
- 양성피드백 강도 신중히 조절 (발산 방지)
- 시냅스 지연 포함 (10-20ms 범위)
- 단기 가소성(촉진/억압) 추가 고려

**표현 정확도**:
- 뉴런당 100-200개 스파이크 필요 (높은 신호-잡음비)
- 인코더/디코더 무게: 전략적 초기화 필수
- 검증: 실제 뉴록과 비교 (상관계수 >0.8)

**확장성**:
- 1,000 뉴런: 표준 노트북, 100ms
- 10,000 뉴런: 고사양 GPU, 1초
- 100,000 뉴런: GPU 클러스터, 실시간 근처

## 6. 핵심 발견사항 정리

### 회로 구현의 보편적 원리

1. **표현의 분산성(Distributed Representation)**
   - 정보는 단일 뉴런에 아님
   - 집단 코딩(population coding)이 노이즈 허용과 유연성 제공
   - NEF의 표현 원리가 이를 정량화

2. **재귀 동역학의 필수성**
   - 지속 활성 = 강한 양성 재귀 피드백
   - 외부 입력 없이 상태 유지 가능
   - 어트랙터 네트워크: 안정적이고 선택적

3. **아키텍처 > 뉴런 수**
   - 1-4개 유닛으로도 의사결정 가능
   - 회로 토폴로지와 연결 패턴이 핵심
   - 최적 크기는 과제마다 다름 (일반 임계값 없음)

4. **다층 메커니즘**
   - 지속 활성: 주 정보 유지
   - 침묵 동역학: 암묵적 정보 저장
   - 시냅스 가소성: 적응적 안정성 조절

### 뇌와 인공 신경망의 간극

| 측면 | 뇌 회로 | 현재 AI |
|------|--------|--------|
| **최적 크기** | 작은 부분최적값(과제별) | 더 크면 더 좋음 경향 |
| **가소성** | 적응적, 여러 시간척도 | 고정(학습 후) |
| **신경조절** | 확산적, 모듈레이터 풍부 | 거의 없음 |
| **효율성** | 극도로 효율적 | 계산량 많음 |
| **실시간성** | 자연스러운 일반 작동 | 배치 처리 최적 |
