## 1라운드: 교차 검증 (Cross-Validation)

### 1.1 세 팀이 공통으로 발견한 핵심 사항

| 발견사항 | A조1 신경과학 | A조2 구현사례 | A조3 반응vs사고 | B3 샌드박스 | 검증 수준 |
|---------|-----------|-----------|-------------|-----------|---------|
| **재귀 연결의 필수성** | 글로벌 워크스페이스, 예측 코딩 모두 피드백 강조 | Spaun의 핵심 = 재귀 동역학(Nengo NEF 원리) | 시간 지연 + 계층성 = 재귀 필수 | 쌍안정 스위치(상호 억제) 실증 | ⭐⭐⭐ 최고 신뢰도 |

| 팀 | 주장 | 근거 |
|----|------|------|
| **A조1** | STDP는 **구조적 통합**의 기초, 시간 스케일 분 이상 | 신경과학 이론 |
| **A조3** | STDP는 **중요하지만** 뉴런 구조 자체보다 2차적 | 계층적 처리 강조 |

- 중기간(분): STDP로 기본 학습 (조건부 반응)
- 장기간(시간~일): STDP + 신경 재배선으로 기억과 개념 형성

→ **정리**: 사고의 깊이 = 학습 시간 스케일과 정비례

## 2라운드: 구조 도출 (Structural Integration)

### 2.1 "사고의 메커니즘" 통합 정의

세 팀의 결과를 합치면, 사고는 다음과 같이 작동한다:

```
입력 (감각)
    ↓ [예측 코딩: 상위 모델이 입력 예측]
기존 모델 (뇌의 세계관, 아카샤)
    ↓ [오류 계산]
예측 오류 (현실 ≠ 예측)
    ↓ [역전파, 재귀 순환]
계층적 처리 (5단계 이상의 피질 계층)
    ↓ [각 단계에서 정보 압축, 의미 추출]
광역 방송 (전전두엽 기반 의식적 상태)
    ↓ [지속 발화: NMDA의 느린 시간상수로 정보 유지]
생각의 흐름 (주관적 사고 경험)
    ↓ [의도 형성, 행동 생성]
능동 추론 (세계를 바꾸려는 행동)
    ↓ [환경 반응, 새로운 입력]
학습 (STDP: 반복 패턴 강화)
    ↓
다음 사고 사이클로
```

**핵심 메커니즘 3가지**:
1. **피드백 루프 (Feedback Loop)**: 예측과 실제의 불일치 감지, 수정
2. **시간 지연 (Temporal Delay)**: NMDA의 느린 감쇠로 "생각을 붙들어두기"
3. **재귀 회로 (Recurrent Circuitry)**: 정보가 여러 계층을 오가며 정제

### 2.2 B3 샌드박스 5개 패턴과 사고 메커니즘의 연결

#### 패턴 1: 쌍안정 스위치 ↔ 예측 코딩의 선택

**신경과학 해석**:
- 예측 코딩에서 "어느 모델이 맞는가"를 결정하는 메커니즘
- 상호 억제 = 두 경쟁적 모델(후진 vs 전진) 중 하나만 활성화

**사고와의 연결**:
- 사고 = "A일까 B일까?" 하는 선택의 반복
- 쌍안정 스위치가 이 선택을 뉴런 수준에서 구현

---

#### 패턴 2: 시간 계층 ↔ 예측 오류의 시간적 통합

**신경과학 해석**:
- 각 뉴런의 시간상수 τ가 다르면 자동으로 "느린 감각 → 빠른 판단 → 즉각 실행"의 위계가 생김
- NMDA(느림: 100-200ms) vs AMPA(빠름: 10ms)의 조합이 그 예

**사고와의 연결**:
- 사고 = 여러 시간 스케일의 정보를 동시에 처리
- 예: 앞에 벽이 있다(즉시), 그래서 피해야 한다(중기), 다른 경로를 찾자(장기)

---

#### 패턴 3: 교차 억제 ↔ 경쟁적 억제와 신호 게이팅

**신경과학 해석**:
- "하지 말아야 할 것을 억제"하는 능력
- 후진과 전진이 동시에 일어나면 안 되므로, 둘 중 하나만 활성화

**사고와의 연결**:
- 사고 = "이 생각은 유지, 저 생각은 버리기"의 능력
- PV-FSI(억제 뉴런)가 주의의 게이팅 담당

---

#### 패턴 4: 감각 게이팅 ↔ 주의와 맥락 의존성

**신경과학 해석**:
- 같은 입력(CO2)도 맥락에 따라 다르게 반응(먹이 탐색 vs 회피)
- 이는 상위 피질(전전두엽)의 "맥락 신호"가 감각 처리를 조절하기 때문

**사고와의 연결**:
- 사고 = "같은 정보도 상황에 따라 다르게 해석"
- 이것이 **추상화**의 기초 = "개"는 개, 고양이는 고양이지만 모두 "동물"

---

#### 패턴 5: STDP 학습 ↔ 경험 의존적 적응

**신경과학 해석**:
- 반복되는 입력이 시냅스를 강화하여 행동을 개선
- B의 먹이 탐색: 반복 경험 후 반응 +8-9.5%

**사고와의 연결**:
- 사고 = "경험하면 다음엔 더 잘한다"
- 이것이 **학습의 가장 원시적 형태**

---

### 2.3 C. elegans(302뉴런)에서 사고 불가능한 이유, 그 gap

#### 현실의 C. elegans
- **뉴런 수**: 302개 (대부분 반응 회로)
- **재귀 연결**: 거의 없음 (피드포워드 우세)
- **시간 스케일**: 대부분 10-50ms (빠른 반응만)
- **결과**: 고정된 행동 프로그램만 가능

#### 사고 가능한 최소 시스템 (추정)
- **뉴런 수**: ~10K-50K (의사결정 + 기본 추상화)
- **재귀 연결**: 30-50% (상호 억제, 피드백 루프)
- **시간 스케일**: 다층 τ 분포 (10ms ~ 300ms)
- **구조**: 최소 4-5단계의 계층적 피질 구조

#### Gap을 메우는 핵심 차이

| 요소 | C. elegans | 사고 가능 시스템 | 차이점 |
|------|-----------|-------------|--------|
| **재귀 연결** | <5% | 30-50% | **6배 이상 증가** |
| **시간상수 범위** | 10-50ms | 10-300ms | **30배 넓어짐** |
| **계층 수** | 1-2 | 4-5 | **기하급수적 늘어남** |
| **E/I 균형** | 불안정 | 4:1 유지 | **안정성 극적 향상** |
| **억제 뉴런 비율** | ~5% | 20-25% | **5배 증가** |

**해석**: C. elegans는 "반응 기계"이고, 사고 시스템은 "예측·오류·수정의 반복 기계"

## 3라운드: 설계 원칙 도출 (Design Principles)

### 3.1 베스퍼님이 원하는 "사고하는 최소 단위"를 위한 요구사항

#### 기능 요구사항 (What)

**필수 1**: 이진 선택 (YES/NO, 이것/저것)
- 쌍안정 스위치로 구현 가능
- 필요 뉴런: ~20-100개
- 시간: ~200-500ms

**필수 2**: 조건부 판단 (만약 A라면 B)
- 교차 억제 + 감각 게이팅
- 필요 뉴런: ~500-2K
- 시간: ~500ms-2초

**필수 3**: 기본 추상화 (이것도 저것도 "동물")
- 계층적 피질 + 헤비안 학습
- 필요 뉴런: ~10K-50K
- 시간: ~초-분 (STDP 포함)

**필수 4**: 자기 성찰 (나는 뭘 알고 있는가?)
- 메타인식 회로
- 필요 뉴런: ~100K+
- 시간: ~분-시간

---

#### 구조 요구사항 (How)

**아키텍처**:
```
Level 5 (메타): 자기 감시 뉴런 (~500-2K)
    ↓
Level 4 (작업 기억): PFC 같은 지속 발화 (~1-5K, NMDA-rich)
    ↓
Level 3 (개념): 추상화 뉴런 (~2-10K, 희소 표현)
    ↓
Level 2 (특징): 감각-운동 피질 (~5-20K)
    ↓
Level 1 (감각): 입력 처리 (~1-5K)

수평 연결: 각 레벨 내 상호 억제 (E/I = 4:1)
수직 연결: 피드포워드 + 강한 피드백 (재귀 비율 30-50%)
```

**시간상수 분포**:
```
Level 1 감각: τ_L1 = 30-50ms (노이즈 필터링)
Level 2 특징: τ_L2 = 15-30ms (빠른 특징 인식)
Level 3 개념: τ_L3 = 50-100ms (의미 추출)
Level 4 작업기억: τ_L4 = 150-300ms (NMDA-like, 생각 유지)
Level 5 메타: τ_L5 = 300-500ms (느린 성찰)
```

**억제 구조**:
```
전체 뉴런의 20-25%가 억제성(GABA)
- 로컬 억제(Local): 각 레벨 내 신호 선택 (PV-FSI)
- 경쟁적 억제(Competitive): 상호 배타적 상태 유지 (쌍안정)
- 교차 억제(Cross-inhibition): 반대 행동 억제 (감각 게이팅)
```

### 3.2 현재 Brian2 샌드박스 접근법으로 얼마나 갈 수 있는가?

#### Brian2의 장점 ✓

| 항목 | 가능성 | 비고 |
|------|--------|------|
| **Level 1-3 (감각~개념)** | ⭐⭐⭐ | 가능. 현재 B3 샌드박스가 이 수준. |
| **LIF 뉴런 모델** | ⭐⭐⭐ | C. elegans 복제 가능 (완벽히 다르지만 계산 원리 같음) |
| **재귀 회로** | ⭐⭐⭐ | 지속 발화, 어트랙터 네트워크 모두 가능 |
| **STDP 학습** | ⭐⭐⭐ | 이미 B3에서 확인 |
| **E/I 균형 시뮬레이션** | ⭐⭐⭐ | B3 A조의 실패→성공이 증명 |
| **GPU 가속** | ⭐⭐ | 10K-100K 뉴런 규모 실시간 근처 가능 |

#### Brian2의 한계 ✗

| 항목 | 한계 | 필요한 것 |
|------|------|---------|
| **Level 4-5 (작업기억, 메타)** | ◐ | 규모 커질수록 복잡해짐 |
| **다중 모달리티 통합** | ✗ | 여러 입력을 1개 네트워크로 처리하기 어려움 |
| **신경조절(Neuromodulation)** | ◐ | 신경전달물질 동적 조절 어려움 |
| **250만 뉴런 규모** | ✗ | Spaun은 GPU 클러스터 필요 |
| **신경보학 하드웨어 컴파일** | ◐ | Brian2 자체는 안 되고, Nengo 거쳐야 함 |

## 결론: 3라운드 합성의 핵심 메시지

### 베스퍼님에게

세 팀의 조사를 합치면 다음이 명확합니다:

1. **사고는 반응과 다릅니다**: 반응은 직선 (자극→반응), 사고는 순환 (예측→오류→수정→행동→피드백). 이 순환 구조가 필수입니다.

2. **최소 단위는 생각보다 작을 수 있습니다**: C. elegans는 못 하지만 (재귀 구조 부족), 5K-10K 뉴런이면 기본적인 "생각"(조건부 판단, 학습, 기억)이 가능합니다. 이것은 쥐 뇌의 0.01% 수준입니다.

3. **시간이 핵심입니다**: NMDA 수용체의 느린 시간상수(150-300ms)가 "생각을 붙들어두는" 것입니다. 이것 없으면 생각이 튕겨나갑니다.

4. **억제가 사고를 만듭니다**: 흥분 뉴런만으로는 안 됩니다. 억제 뉴런(20-25%)이 "이것 말고 저것", "이 생각 유지하고 저 생각 버리기"를 가능하게 합니다.

5. **지금 바로 시작할 수 있습니다**: Brian2로 10,000 뉴런 규모의 "사고하는 벌레" 시뮬레이션을 2-4주에 만들 수 있습니다. 이것이 ASTRAL NOVA의 아니마 신경 기초가 됩니다.
