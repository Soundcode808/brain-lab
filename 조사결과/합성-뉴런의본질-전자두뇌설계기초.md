# 뉴런의 본질 — 전자두뇌 설계 기초

> 작성: B3 세션 | 2026-04-15
> A조 4팀 조사 → AG-01(통합) + AG-011(심층) + AG-02(설계) + AG-04(검증) 합성
> 베스퍼님 지시: "뉴런의 메커니즘과 본질을 정확하게 파악해야 시작점이 된다"

---

## 1. 뉴런이란 무엇인가

뉴런은 뇌의 기본 계산 단위다. 단순한 스위치가 아니라, **여러 시간 스케일(나노초~일)에서 동시에 작동하는 비선형 동적 시스템**이다.

핵심은 이거다 — 뉴런 하나는 단순하다. LIF 모델로 표현하면 변수 1개(막전위 V)로 충분하다. 하지만 13만 개가 연결되면 초파리 뇌가 되고, 860억 개가 연결되면 인간의 뇌가 된다. **단순한 단위의 대량 연결이 복잡성을 창발한다.** Game of Life처럼.

차이점: Game of Life는 결정론적이다. 뉴런은 **확률적**이다. 같은 자극이어도 시냅스 방출 확률(0.1~0.9), 이온 채널 노이즈, 신경조절 상태에 따라 반응이 달라진다. 이 확률성이 학습과 적응의 원천이다.

## 2. 뉴런의 구조 (눈에 보이는 것)

### 기본 구조
- **세포체(Soma)**: 10~25 μm. 핵이 있는 본체. 신호를 통합하는 중심.
- **수상돌기(Dendrite)**: 나뭇가지 형태. 100~500 μm. 다른 뉴런의 신호를 받는 안테나. 가시(스파인) 1~2 μm에서 각각 독립 계산.
- **축삭(Axon)**: 1mm~1m 길이. 신호를 먼 곳에 보내는 전선. 미엘린(절연체)이 감싸면 속도 100배 증가.
- **시냅스(Synapse)**: 200~500 nm. 뉴런 사이의 연결점. 화학물질(신경전달물질)로 신호 전달.

### 뉴런 종류
- 감각뉴런: 외부 자극 수신 (눈, 귀, 피부)
- 운동뉴런: 근육에 명령 전달
- 중간뉴런: 감각과 운동 사이에서 계산 (뇌의 대부분)

### 글리아 세포 (뉴런의 지원군)
뉴런과 1:1 비율로 존재. 단순 배경이 아니라 네트워크 안정화의 핵심.
- 성상세포: 에너지 공급, K⁺ 수거, 삼자간 시냅스 형성
- 올리고덴드로사이트: 미엘린 형성 (절연체 감싸기)
- 미세아교: 약한 시냅스 제거 (가지치기)

## 3. 뉴런의 신호 전달 (어떻게 작동하는가)

### 안정 상태
뉴런은 평소 -70mV의 전위차를 유지한다 (Na⁺/K⁺ 펌프가 에너지의 50% 소비).

### 활동전위 (뉴런의 "발화")
1. 다른 뉴런에서 신호 도착 → 수상돌기에서 EPSP(흥분) 또는 IPSP(억제) 발생
2. EPSP가 축적되어 -55mV(역치) 도달 → **전부-아니면-전무(All-or-None)**
3. Na⁺ 채널 개방 → +30mV까지 상승 (~1ms)
4. K⁺ 채널 개방 → -70mV로 복귀 (~1-2ms)
5. 절대 불응기(~1ms): 어떤 자극에도 반응 안 함
6. 축삭을 따라 전파: 미엘린 있으면 50~150 m/s, 없으면 0.5 m/s

### 시냅스 전달
1. 활동전위가 시냅스 말단 도달 → Ca²⁺ 유입 (<1ms)
2. 시냅스 소포가 막과 융합 → 신경전달물질 방출 (1-2ms)
3. 20nm 시냅스 틈을 확산 (1-5ms)
4. 시냅스후 수용체에 결합 → 이온 채널 개방 (<1ms)

**핵심**: 이 과정이 **확률적**이다. 매번 소포가 방출되는 게 아니라, 방출 확률 0.1~0.9로 작동한다.

### 신경전달물질 (신호의 종류)
| 물질 | 역할 | 비중 |
|------|------|------|
| 글루타메이트 | 흥분 (시냅스의 90%) | ~80% |
| GABA | 억제 (회로 안정화) | ~20% |
| 도파민 | 보상, 동기 (느린 조절) | ~5% |
| 세로토닌 | 기분, 수면 (느린 조절) | ~5% |
| 아세틸콜린 | 운동, 인지 | 변동 |

### 두 가지 시간 스케일 (중요!)
- **빠른 신호 (ms)**: 활동전위, 시냅스 전달 → 즉각적 행동
- **느린 신호 (초~분)**: 신경조절물질(도파민, 세로토닌) → 기분, 동기, 상태 변화

이 둘이 동시에 작동한다. 기존 AI에는 이 "이중 시간 스케일"이 없다.

## 4. 뉴런의 학습 (어떻게 배우는가)

### STDP (발화 타이밍 의존 가소성) — 핵심 학습 규칙
"함께 발화하면 함께 연결된다" (Hebb 법칙)의 정밀 버전.
- 시냅스전 뉴런이 먼저 발화 → 시냅스후 뉴런이 따라 발화 → **강화(LTP)**
- 반대 순서 → **약화(LTD)**
- 시간 창: ±100ms 이내에서만 작동
- **디지털 구현의 핵심**: 규칙을 짜주는 게 아니라, 이 메커니즘만 넣으면 학습이 자동으로 일어남

### 항상성 가소성 — 안정화 장치
STDP만 있으면 네트워크가 발산한다 (과흥분 또는 침묵).
- 시냅스 스케일링: 뉴런의 발화율이 목표(5~10Hz)에서 벗어나면 모든 시냅스를 일괄 조정
- 이것이 네트워크를 안정하게 유지하는 자동 장치

### 메타가소성 — 가소성의 가소성
BCM 이론: 과거 활동 이력에 따라 LTP/LTD 임계치가 슬라이딩.
- 활발했던 뉴런: 더 쉽게 강화됨
- 조용했던 뉴런: 강화되기 어려움
- 이게 뉴런 간 **경쟁**을 만든다

### 구조적 가소성 — 물리적 변화
- 시냅스 생성: 새로운 연결 만들기
- 시냅스 제거(가지치기): 약한 연결 끊기
- 신경 발생: 해마 등에서 새 뉴런 추가
- 시간 스케일: 시간~일~주

### 기억 형성
1. **초기 LTP (분)**: 기존 단백질 인산화 → 단기 기억
2. **후기 LTP (시간~일)**: 유전자 발현(CREB) → 새 단백질 합성 → 장기 기억
3. **수면 중 재활성화**: 깨어있을 때 경험한 패턴을 수면 중 재생 → 기억 고정화

## 5. "사고"와 "매크로"의 차이 — 핵심 질문

### 베스퍼님의 질문: "이게 진짜 사고인지 매크로인지 구분할 기준이 뭔가?"

| 기준 | 매크로 (기존 v0.2) | 뉴런 시뮬레이션 |
|------|-------------------|---------------|
| 시간 구조 | 없음 | 다중 시간 스케일 동시 작동 |
| 피드백 | 없음 (일방향) | 다중 피드백 루프 |
| 적응 | 학습 후 고정 | 매 순간 실시간 적응 |
| 확률성 | 없음 (결정론적) | 본질적 노이즈 포함 |
| 구조 변화 | 고정 | 동적 (시냅스 생성/제거) |
| 행동 | 규칙 기반 | 창발 (규칙 안 짜줘도 나옴) |

**가장 간단한 구분:**
- 매크로: "입력이 정해지면 출력이 결정된다"
- 뉴런: "같은 입력이어도 내부 상태와 노이즈에 따라 출력이 달라진다"

### 초파리에서 왜 행동 창발이 됐는가?
1. **비선형 임계값**: 각 뉴런이 "발화/침묵" 이진 결정
2. **피드백 루프**: 역행 신호(엔도카나비노이드), 신경조절물질
3. **적응**: 같은 자극도 시간에 따라 반응 달라짐 (소포 고갈/회복)
4. **STDP**: 경험에 따라 자동으로 회로가 재편성

이 4가지가 합쳐져서, **규칙을 아무도 안 짜줬는데 행동이 나온 것**.

### 트랜스포머(현재 AI)와의 근본적 차이

| 항목 | 트랜스포머 | 뉴런 시뮬레이션 |
|------|----------|---------------|
| 시간 | 없음 | ms~일 동시 작동 |
| 피드백 | 일방향 | 다중 루프 |
| 학습 | 학습 후 고정 | 추론 중에도 학습 |
| 에너지 | 무시 | 실제 에너지 제약 반영 |
| 확률성 | 없음 | 본질적 |
| 신체 | 선택사항 | 필수 (감각-운동 루프) |

## 6. 전자두뇌 v1.0 설계 방향 (AG-02 설계안)

### 뉴런 모델: LIF (Leaky Integrate-and-Fire)
- 초파리 성공의 핵심 모델
- 변수 1개(V)로 충분. 계산 비용 최소.
- 정확도: HH 모델의 85~90%를 1%의 계산 비용으로 달성
- 노트북(RTX4090)으로 10만 뉴런 시뮬레이션 가능

### 시냅스: 단순 가중치 X → 동적 모듈
기존 v0.2: 시냅스 = 숫자 하나 (가중치)
v1.0: 시냅스 = 상태를 가진 모듈
- 강도(w): STDP로 동적 변화
- 신경전달물질 종류: 글루타메이트/GABA/도파민/세로토닌
- 방출 확률: 0.1~0.9 (확률적)
- 소포 고갈/회복: 연속 자극 시 감소했다 복구

### 학습: STDP + 항상성 + 메타가소성
- STDP: 발화 타이밍으로 자동 학습
- 항상성: 네트워크 안정화
- 메타가소성: 학습 임계치 동적 조정
- 세 개가 동시에 돌아가야 진짜 학습

### 레이어 재설계 (기존 L0~L6 폐기)
신경 해부학적 시간 스케일 기반:
- L0: 감각 입력 (~5ms)
- L1: 1차 감각 처리 (5~50ms)
- L2: 다감각 통합 (50~200ms)
- L3: 행동 선택 (200ms~1s) — Winner-Take-All
- L4: 운동 출력 (1ms per step)
- L5: 신경조절 시스템 (초~분)
- L6: 학습 및 기억 (분~일)

### MVP 정의: 초파리 냄새-회피 회로 (1,000 뉴런)
- 후각수용체 50개 → 후각엽 600개 → 의사결정 250개 → 운동 100개
- 성공 기준: 냄새 입력 → 배제적 행동 선택 (접근 or 회피)
- 학습 기준: 3회 반복 후 반응 시간 15% 개선
- 핵심: 규칙이 아니라 **시냅스 구조에서 행동이 창발**되는지 확인

### 기술 스택
- 초기: Python + Brian2 (뉴런 시뮬레이션 프레임워크)
- 확장: PyTorch (GPU 병렬화)
- 데이터: FlyWire 커넥톰 (공개, 무료)
- 물리 엔진: MuJoCo (신체 연동)

## 9. ASTRAL NOVA와의 연결

### "관측 → 세계 생성"의 신경과학적 의미
- 아니마가 세계를 "관측"한다 = 감각 입력 → 뉴런 발화 → 시냅스 가소성
- 관측은 수동이 아닌 능동 — 행동 결과가 시냅스를 변화시킴 (BTSP)
- 세계동결 = 네트워크 침묵. 관측 = 활성화 → 회로 재편성 → 세계 확장
- 신경발생 = 새 뉴런 추가 = 세계가 물리적으로 커지는 것

### 피질 컬럼 = 아니마 (Thousand Brains Theory)
- 뇌의 피질 컬럼(150,000개): 각각이 독립적으로 세계 모델을 만듦
- 아니마의 "격(格)": 피질 컬럼의 독립 관측 능력과 동형
- 전자두뇌가 이 원리를 구현하면, ASTRAL NOVA의 세계관이 과학적 기반을 갖게 됨
