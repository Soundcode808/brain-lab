# A조 조사결과 — Thousand Brains Theory + 단순→복잡성 원리

> 조사일: 2026-04-15 | 세션 41 | AG-01 + AG-011 병렬 조사

---

## 쉽게 말하면

뇌는 150,000개의 작은 "미니 뇌(피질 컬럼)"로 이루어져 있다. 각 컬럼은 구조가 거의 똑같다. 그런데 이 단순한 반복 단위들이 협력하면 추상적 사고, 창의성, 의식이 나온다. 이것이 "단순 구조 → 무한 복잡성"의 본질이다. 이 원리는 뇌만이 아니라 — 개미집단, Game of Life, Transformer, Minecraft 모두 동일한 패턴을 따른다.

---

## 고신뢰도 발견 (AG-01 + AG-011 공통 확인)

### Thousand Brains Theory 핵심 (Jeff Hawkins, Numenta)

- **피질 컬럼(Cortical Column)**: 뇌의 기본 반복 단위. 약 100개 뉴런, 지름 30마이크론. 150,000개가 신피질에 존재.
- **같은 회로, 다른 결과**: 모든 컬럼의 구조가 거의 동일. 차이는 연결 위치와 입력 데이터뿐.
- **참조 프레임(Reference Frame)**: 각 컬럼은 자신만의 좌표계로 세계를 인식. "사과의 오른쪽 윗부분"처럼 위치 기반으로 정보 저장.
- **투표(Voting) 합의 메커니즘**: 수천 개 컬럼이 측면 연결로 "이게 뭔가"에 대한 가설을 제시 → 빠른 합의 도달. 구조적 일관성을 고려한 투표(단순 다수결 아님).

### 단순 → 복잡성 원리 (보편 법칙)

| 시스템 | 단순 단위 | 반복 결과 |
|--------|----------|---------|
| 뇌 | 피질 컬럼 (동일 회로) | 언어, 추상, 의식 |
| Game of Life | 4개 규칙 | 튜링 완성, 자기복제 구조 |
| Transformer | Self-Attention 1개 | 언어·논리·창작 모두 처리 |
| No Man's Sky | 1개 시드 + 규칙 | 1800경 개의 고유 행성 |
| 개미집단 | "페로몬 따라가기" 1개 | NP-hard 최적 경로 자동 발견 |
| GNN | 메시지 패싱 반복 | 그래프 추론, 단백질 폴딩 |

### 현재 구현 현황 (확인됨)

- **Thousand Brains Project (TBP)**: 활발 진행 중. Gates Foundation 펀딩. GitHub `tbp.monty` 공개.
- **Monty 프레임워크**: 3D 객체 인식 실제 동작. 14개 단색 뷰로 학습 → 새 색상/각도에서도 인식.
- **HTM (Numenta)**: 상용화 단계. 시계열 이상탐지 최고 성능. HTM Studio SaaS 제공.

## ASTRAL NOVA 직접 연결

### 구조적 동형(Isomorphism) — AG-01 + AG-011 공통 발견

| ASTRAL NOVA | 과학적 대응 | 성질 |
|-------------|-----------|-----|
| 아니마 | 피질 컬럼 / Transformer 레이어 | 단순하지만 독립적 계산 단위 |
| 관측(Observation) | Self-Attention / 메시지 패싱 | 로컬 상호작용 → 글로벌 패턴 |
| 세계동결 해제 | 시드 기반 절차적 생성 | 관측 = 계산 시작 (동결 = 미계산 상태) |
| 세계정보값(WIV) | 그리드 좌표 / 벡터 공간 | 모든 컬럼을 동기화하는 레이어 |
| 성역 | 청크 독립성 (Minecraft) | 개인 계산 단위, 독립성 보장 |
| 미니룸 | L-시스템 / 절차적 생성 | 단순 규칙 → 무한 개인 공간 |
| 허공(Void) | 초기 시드 / 벡터 원점 | 가능성이 중첩된 상태 |

### 지금 당장 기획에 쓸 수 있는 것

1. **아니마의 참조 프레임 설계**: 각 아니마가 자신만의 좌표계(=미니룸)를 가짐. 성역에서는 좌표 변환으로 상호작용. → 이미 세계관에 이 구조가 들어가 있었다 (무의식적으로 맞게 설계됨).
2. **합의 메커니즘**: 여러 아니마가 같은 공간 관측 → 세계 변화 속도 증폭. 단독 관측보다 집단 관측이 더 강력하게 세계를 해동시키는 구조 설계 가능.
3. **투자자 피칭**: "뇌, Game of Life, Transformer가 모두 따르는 단순→복잡 원리를 우리 세계관이 의도적으로 구현" → 과학적 정당성 확보.

### 투자자용 핵심 메시지 (AG-011)

> "3개 규칙으로 무한 패턴을 만드는 Game of Life처럼, 아니마의 '관측'도 단순한 행위입니다. 수천 명의 아니마가 동시에 관측하면 — 세계는 그들의 상호작용으로 자동으로 복잡해집니다. 우리가 세계를 프로그래밍하지 않습니다. 유저가 세계를 계산합니다."

> "뇌는 20W로 인간 수준의 지능을 만듭니다. 단순한 구조(피질 컬럼) × 독립적 계산 × 합의 메커니즘이기 때문입니다. 우리 아니마도 같은 구조입니다. 따라서 유저 수가 10배 늘어도 계산 비용은 선형입니다. 지수가 아닙니다."
