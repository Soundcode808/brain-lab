## 1. 안정막전위(Resting Potential)

### 원리와 형성
뉴런은 세포 내외의 이온 농도 차이로 인해 막 전위(membrane potential)를 유지한다. 일반적인 뉴런의 축삭 언덕(axon hillock)에서 휴지 전위는 **-70mV** 수준이다.

### Na+/K+ ATPase 펌프
안정막전위를 유지하는 핵심 메커니즘은 나트륨-칼륨 펌프(Na+/K+-ATPase)이다:

- **작동 방식:** 1개의 ATP를 분해하여 3개의 Na+ 이온을 세포 외부로, 2개의 K+ 이온을 세포 내부로 펌프한다
- **전기생성성(Electrogenic):** 3Na+ 배출 vs 2K+ 유입의 불균형으로 인해 -10mV 정도 추가 음전위 생성
- **에너지 비용:** 뉴런 세포 에너지의 약 50%를 소비
- **누출 채널(Leak channels):** Na+와 K+는 농도 기울기를 따라 뒷막으로 누출되며, 펌프는 이를 보상

### 이온 농도 (세포 내/외)
| 이온 | 세포 내 (mM) | 세포 외 (mM) | 네른스트 전위 |
|-----|------------|-----------|-----------|
| K+ | 140 | 5 | -90mV |
| Na+ | 15 | 150 | +60mV |
| Cl- | 5-15 | 110 | -70mV |

### 디지털 구현 핵심 파라미터
- `restingPotential = -70 mV`
- `naOutfluxRate = 3` (per ATP)
- `kInfluxRate = 2` (per ATP)
- `leakConductance_Na` = f(V) [전압 의존]
- `leakConductance_K` = f(V) [전압 의존]
- `ATPAvailable` = 에너지 풀, 펌프 작동 제한

## 2. 활동전위(Action Potential)

### 단계별 메커니즘

#### 2.1 정지 상태 (~-70mV)
- 안정막전위 유지
- 전압 의존 채널 대부분 닫혀 있음

#### 2.2 탈분극 상태 (Depolarization)
- 자극이 막전위를 -70mV에서 더 양전하 방향으로 변화시킴
- 전압의존성 Na+ 채널이 열리기 시작
- 양의 피드백: Na+ 유입 → 탈분극 → 더 많은 Na+ 채널 개방

#### 2.3 역치(Threshold) 현상
- **역치 전위:** 약 -55mV
- All-or-None 법칙: 역치에 도달하면 활동전위는 **반드시** 발생
- 역치 이하: 활동전위 없음 (국소 전위만 존재)
- 역치 초과: 자극 강도와 관계없이 동일한 크기의 활동전위 발생

#### 2.4 상향 구간(Upstroke, ~1ms)
- 전압의존 Na+ 채널 최대 개방
- 막전위는 -55mV에서 +30mV 부근까지 상승
- Na+ 내유입 속도 >> 외유출 속도

#### 2.5 재분극(Repolarization, ~1-2ms)
- Na+ 채널 불활성화 (inactivation): h게이트 닫힘
- 전압의존 K+ 채널 천천히 개방 (Na+ 채널보다 느림)
- K+ 외유출 시작 → 탈분극 역전

#### 2.6 과분극(Hyperpolarization, ~1-2ms)
- K+ 채널가 여전히 개방되어 있음
- 막전위는 -70mV를 넘어 -90mV까지 저하
- K+ 채널 천천히 닫힘 → 다시 -70mV로 복귀

### 불응기(Refractory Period)

#### 절대 불응기(Absolute Refractory Period)
- **지속 시간:** ~1ms
- **원인:** Na+ 채널이 불활성화(h게이트 닫힘)
- **특성:** 어떤 자극도 새로운 활동전위 유발 불가능
- **기능:** 활동전위 단일성 보장, 신호 역전파 방지

#### 상대 불응기(Relative Refractory Period)
- **지속 시간:** ~10-100ms
- **원인:** 막전위가 과분극 상태, 더 큰 자극 필요
- **특성:** 평상시보다 강한 자극이면 활동전위 발생 가능
- **기능:** 신경 발화 빈도 제어, 뉴런 안정화

### 활동전위의 형태
```
전위(mV)
  +40 |     /\
       |    /  \
    0 |   /    \___
       |  /         \___
  -55 |_/________________
       |------1ms------1ms----10ms----
       휴지   상향    재분극   과분극
```

### 디지털 구현 핵심 파라미터
- `threshold = -55 mV`
- `peakVoltage = +30 mV`
- `hyperpolarizationMin = -90 mV`
- `Na_channelOpenRate` = f(V, t) [Hodgkin-Huxley 모델]
- `Na_channelInactivationRate` = f(V, t)
- `K_channelOpenRate` = f(V, t)
- `absoluteRefractoryPeriod = 1 ms`
- `relativeRefractoryPeriodEnd = ~100 ms`
- `allOrNoneThreshold = true` [디지털 임계값]

## 3. 전도(Conduction)

### 연속전도(Continuous Conduction) vs 도약전도(Saltatory Conduction)

#### 연속전도(Unmyelinated Axons)
- **미엘린 유무:** 없음 (또는 희박)
- **신호 전파:** 축삭 전체 길이에서 연속적으로 활동전위 발생
- **전도 속도:** 약 0.5~2 m/s (비교: 시속 2~7km)
- **에너지 효율:** 낮음 (전체 막에서 이온 펌프 필요)
- **생물학적 예:** 부교감신경, 일부 감각신경

#### 도약전도(Myelinated Axons)
- **미엘린 유무:** Oligodendrocytes(중추신경) 또는 Schwann cells(말초신경)가 형성
- **구조:** 
  - **수초(Myelin sheath):** 미엘린 지질이중층으로 축삭 감싸기
  - **랑비에 결절(Nodes of Ranvier):** 미엘린 부재 구간 (~1μm), 약 0.2-2mm 간격
- **신호 전파:** 활동전위가 한 결절에서 다음 결절로 "도약"
  - 결절 간 구간에서는 근처의 활동전위 이웃 결절에 영향
  - 국소 전류(local current)가 다음 결절을 탈분극
- **전도 속도:** 약 50~150 m/s (비교: 시속 180~540km)
- **속도 증가 배수:** 연속전도 대비 50~100배
- **에너지 효율:** 높음 (결절에서만 이온 펌프 작동 필요)

### 전도 속도에 영향을 주는 요소

| 요소 | 영향 | 설명 |
|------|------|------|
| **미엘린 존재** | 매우 큼 | 도약전도 활성화, 50-100배 증가 |
| **축삭 직경** | 중간 | 직경 ∝ 속도 (저항 감소) |
| **온도** | 중간 | 높을수록 이온 채널 활성 증가 |
| **결절 간 거리** | 작음 | 최적화된 간격 있음 |
| **Na+ 채널 밀도** | 중간 | 결절에서 채널 농도 높음 |

### 케이블 방정식(Cable Equation)
신경 축삭을 동축 케이블로 모델링하면:

```
τ × dV/dt = λ² × d²V/dx² - V

여기서:
τ = 시간 상수 (membrane capacitance × resistance)
λ = 공간 상수 = √(r_m / r_i)
     r_m = 막 비저항, r_i = 축삭 내 비저항
```

**물리적 의미:**
- 공간 상수 λ가 크면 (굵은 축삭, 높은 r_m) → 신호 원거리 전파 가능
- 공간 상수 λ가 작으면 (가는 축삭, 낮은 r_m) → 신호 빠르게 감쇠

### 디지털 구현 핵심 파라미터
- `myelinated = bool` [true면 도약전도]
- `nodeDistance = 0.2~2 mm` [랑비에 결절 간 거리]
- `cableResistance_axial` = r_i [축삭 내 저항]
- `membraneResistance` = r_m [막 저항]
- `spatialConstant_lambda = √(r_m / r_i)`
- `timeConstant_tau` = C_m × r_m
- `conductionVelocity = f(diameter, myelination, temperature)`
  - 도약: ~100 m/s (기본값)
  - 연속: ~1 m/s (기본값)

## 4. 시냅스 전달(Synaptic Transmission)

### 4.1 화학적 시냅스(Chemical Synapses)

시냅스는 다음 세 부분으로 구성:
1. **시냅스전 말단(Presynaptic terminal):** 축삭 팽대부
2. **시냅스틈(Synaptic cleft):** 20nm 정도 간격
3. **시냅스후 구조(Postsynaptic structure):** 수상돌기 또는 세포체

#### 신경전달물질 방출 과정

**Step 1: 포장(Packaging)**
- 신경전달물질 분자들이 시냅스전소포(synaptic vesicles, ~40nm 직경)에 수천 개 함께 포함
- 시냅스전 말단에 약 300~1000개의 소포 존재

**Step 2: 도킹(Docking)**
- 소포가 활성화 영역(active zone)의 미출시 사이트(release site)에 부착
- SNARE 단백질 복합체 형성 시작

**Step 3: 프라이밍(Priming)**
- ATP를 소모하여 소포 융합 준비 (fusion competent 상태)
- 소포와 세포막 사이 거리 ~0.5-1nm

**Step 4: 활동전위 도달**
- 축삭 말단 도착한 활동전위 → 전압의존 Ca2+ 채널 개방
- Ca2+ 농도 급증: ~100nM → 1-10μM (100배)

**Step 5: 외세포방출(Exocytosis)**
- Ca2+이 시냅토태그민(synaptotagmin)에 결합
- SNARE 단백질이 mem

brane fusion 촉발
- 소포막과 세포막이 합쳐지면서 신경전달물질 분자 방출
- 약 1000개 분자가 동시에 방출 (1 quantum, 기본 단위)

**Step 6: 재흡수(Reuptake)**
- 신경전달물질이 시냅스틈에서 확산
- 재흡수 단백질(reuptake transporter)을 통해 시냅스전 말단으로 회수
- 또는 시냅스후 세포 또는 글리아세포가 포획

**Step 7: 재포장(Recycling)**
- 회수된 신경전달물질이 새로운 소포에 재포장
- 프라이밍 후 다시 도킹 위치로

#### 시냅스 전달의 시간 스케일
| 단계 | 시간 |
|------|------|
| Ca2+ 채널 개방 | <1ms |
| Ca2+ 확산 | <1ms |
| Exocytosis (소포 융합) | 1-2ms |
| 신경전달물질 확산 | 1-5ms |
| 수용체 결합 | 1-10ms |
| 채널 개방 | 1-20ms |

### 4.2 전기적 시냅스(Electrical Synapses)

- **구조:** 갭 정션(gap junction)을 통한 직접 세포 간 연결
- **특성:** 
  - 화학적 시냅스보다 빠름 (지연 <1ms)
  - 양방향 신호 전달 가능
  - 이온과 작은 분자 직접 통과
- **생물학적 분포:** 뇌에서는 드문, 망막과 심장에서 흔함
- **역할:** 신경 네트워크 동기화

### 디지털 구현 핵심 파라미터
- `vesicleSize = ~40 nm`
- `vesiclesPerTerminal = 300-1000`
- `neurotransmitterPerVesicle = ~1000-5000` [분자 수]
- `quantumSize = 1` [기본 방출 단위]
- `ca2pChannelOpenRate` = f(V)
- `ca2pInfluxRate` = f(time, Ca2p_concentration)`
- `exocytosisLatency = 1-2 ms`
- `synapticCleftWidth = 20 nm`
- `transmitterDiffusionRate` = D [확산 계수]
- `receptorBindingRate` = k_on [시간 의존 확률]
- `receptorUnbindingRate` = k_off
- `reuptakeRate` = f(concentration, transporter_density)
- `vesicleRecyclingTime = 30-60 s`

## 5. 신경전달물질(Neurotransmitters)

### 주요 신경전달물질 분류

#### 5.1 아미노산 신경전달물질

**글루타메이트(Glutamate)**
- **분류:** 흥분성
- **역할:** 
  - 중추신경계 시냅스 전달의 **90% 이상** 담당
  - 학습과 기억의 기초
  - 장기 강화(LTP, long-term potentiation) 관여
- **수용체:** AMPA, NMDA, kainate 타입
- **정상 농도:** 시냅스틈 ~1-2μM
- **병리:** 과잉 흥분성(excitotoxicity) → 신경 손상

**GABA (γ-Aminobutyric Acid)**
- **분류:** 억제성
- **역할:**
  - 중추신경계 주요 억제성 신경전달물질
  - 신경 회로 안정화
  - 항불안, 항경련, 항우울 작용
- **수용체:** GABA_A (이온채널), GABA_B (G단백질)
- **정상 농도:** 시냅스틈 ~1-5μM
- **임상 응용:** 벤조디아제핀(GABA 증강)

**글리신(Glycine)**
- **분류:** 억제성
- **역할:** 척수 신경 억제
- **분포:** 척수, 뇌간

#### 5.2 모노아민(Monoamine) 신경전달물질

**도파민(Dopamine)**
- **분류:** 흥분성/조절성
- **역할:**
  - 운동 제어
  - 보상 회로 중추
  - 동기, 주의, 학습
  - 감정 조절
- **주요 경로:** 중뇌 → 선조체(보상), 전전두엽(인지)
- **병리:** 부족 → 파킨슨병; 과잉 → 정신분열증 가설

**세로토닌(Serotonin, 5-HT)**
- **분류:** 조절성
- **역할:**
  - 기분 조절
  - 수면-각성 주기
  - 통증 인식
  - 온도 조절
  - 소화기능
- **주요 경로:** 등쪽 라파핵 → 뇌 전역
- **병리:** 부족 → 우울증; 과잉 → 세로토닌 증후군

**노르에피네프린(Norepinephrine)**
- **분류:** 흥분성/조절성
- **역할:**
  - 각성 수준 제어
  - 주의 집중
  - 스트레스 반응
  - 혈압/심박수 조절
- **주요 경로:** 청반핵(locus coeruleus) → 뇌 전역

#### 5.3 콜린성 신경전달물질

**아세틸콜린(Acetylcholine)**
- **분류:** 흥분성
- **역할:**
  - 첫 발견 신경전달물질
  - 신경근 접합부 → 근육 수축 지시
  - 중추신경: 학습, 기억, 주의
- **수용체:** 니코틴성(빠른), 무스카린성(느린)
- **병리:** 부족 → 알츠하이머병

### 신경전달물질의 분류 체계
```
신경전달물질
├─ 흥분성 (Excitatory)
│  ├─ Glutamate
│  ├─ Acetylcholine
│  └─ Dopamine (맥락 의존)
│
├─ 억제성 (Inhibitory)
│  ├─ GABA
│  ├─ Glycine
│  └─ (때로는 Dopamine)
│
└─ 조절성 (Neuromodulatory)
   ├─ Serotonin
   ├─ Norepinephrine
   ├─ Histamine
   └─ Neuropeptides
```

### 신경전달물질과 수용체의 상호작용
- **친화도(Affinity):** 신경전달물질이 수용체에 얼마나 잘 결합하는가
- **효율성(Efficacy):** 수용체 활성화 후 신호 강도
- **선택성(Selectivity):** 특정 수용체 타입에 대한 선호도
- **포화도(Saturation):** 수용체가 모두 활성화된 상태

### 디지털 구현 핵심 파라미터
- `transmitterType = ENUM{Glutamate, GABA, Dopamine, ...}`
- `transmitterConcentration = [float]` (μM)
- `receptorAffinity = K_d` [나노몰 범위]
- `receptorEfficacy = β` [0-1 정규화]
- `bindingRate_on = k_on` [per ms]
- `unbindingRate_off = k_off` [per ms]
- `transmitterClass = ENUM{Excitatory, Inhibitory, Modulatory}`
- `halfLife = [ms]` [신경전달물질 분해 시간]

## 6. 시냅스후 전위(Postsynaptic Potential)

### 6.1 흥분성 시냅스후 전위(EPSP)

**정의:** 시냅스후 세포의 막전위가 탈분극(양전하 방향)되는 현상

**메커니즘:**
- 글루타메이트 또는 아세틸콜린 같은 흥분성 신경전달물질이 수용체 결합
- 양이온(주로 Na+) 채널 개방
- Na+ 외유입 → 막전위 -70mV → -50~-60mV (약 10-20mV 상승)
- **크기:** 1mV~수십 mV
- **지속 시간:** 15-100ms

**전형적 예:**
```
시냅스후 전위(mV)
      0 |    ___
        |   /   \
   -50 |__/     \___
        |----------| 20ms
```

### 6.2 억제성 시냅스후 전위(IPSP)

**정의:** 시냅스후 세포의 막전위가 과분극(음전하 방향)되는 현상

**메커니즘:**
- GABA 또는 글리신이 수용체 결합
- 음이온(Cl-) 채널 또는 K+ 채널 개방
- Cl- 내유입 또는 K+ 외유출 → 막전위 -70mV → -80~-90mV (약 10-20mV 저하)
- 또는 하이퍼폴라이제이션 → 활동전위 발생 어려워짐

**억제 메커니즘:**
1. **시냅스전 억제(Presynaptic inhibition):** 그다음 뉴런의 신경전달물질 방출 감소
2. **시냅스후 억제(Postsynaptic inhibition):** 활동전위 유발 더 어렵게

### 6.3 공간적 합산(Spatial Summation)

**정의:** 수상돌기 위 **서로 다른 위치**에서 온 EPSP/IPSP가 합산

**예시:**
```
시냅스 A (근처)     시냅스 B (먼곳)
   EPSP (+10mV)      EPSP (+8mV)
           \                /
            \              /
             모두 솜뮬탕(+18mV) 가능
             (선형 합산이 아닐 수도)
```

**공간적 합산의 특성:**
- 동시에 활성화되는 근처 시냅스들의 효과 증대
- 수상돌기 거리에 따라 감쇠 (케이블 방정식)
- 클러스터된 입력이 더 효과적
- **소뮬탕(Summation):** 각 EPSP의 단순 합이 아니라 비선형 (뒤에서 논의)

### 6.4 시간적 합산(Temporal Summation)

**정의:** **같은 시냅스**에서 생성된 작은 EPSP/IPSP가 시간적으로 가깝게 도착하여 합산

**조건:** 두 자극 사이 간격 < EPSP 지속 시간 (대개 15-100ms)

**예시:**
```
시냅스 A에서 2회 자극
|
첫 번째 EPSP: 
   _____
  /     \___
-50 -----------  (최대값 -50mV)

두 번째 EPSP (5ms 후): 
         _____
        /     \___
-50 --/
-40             (2번째가 1번째 아직 진행 중 → 합산)

합산된 결과:
   _________
  /         \___
-30               (더 큰 탈분극)
```

**시간적 합산 효율:**
- 간격이 짧을수록 (< 10ms) 합산 효율 높음
- 간격이 길수록 (> 100ms) 각각 독립적으로 작용
- **적분 시간:** ~50-100ms

### 6.5 EPSP/IPSP 비율과 행동 출력

일반적인 뉴런은 **수천 개**의 시냅스 입력을 받음:

**뉴런의 통합 규칙:**
```
V_soma = ∑(흥분성 입력 수 × 강도) - ∑(억제성 입력 수 × 강도)

IF V_soma > threshold THEN
    발화 (Action Potential)
ELSE
    침묵
```

**흥분-억제 균형(E-I Balance):**
- 건강한 뇌: 흥분성 ~ 억제성 (약간의 불균형)
- 불균형 → 경련(과흥분) 또는 마비(과억제)

### 디지털 구현 핵심 파라미터
- `EPSP_amplitude = 0-20 mV` (context 의존)
- `EPSP_duration = 15-100 ms`
- `IPSP_amplitude = 0-20 mV` (음수)
- `IPSP_duration = 15-100 ms`
- `spatial_decay = exp(-distance / λ)` [공간 상수]
- `temporal_integration_window = 50-100 ms`
- `summation_linearity = β` [0=완전 선형, 1=완전 비선형]
- `threshold_for_spike = -50 to -40 mV` (맥락 의존)

## 7. 시냅스 통합(Synaptic Integration)

### 7.1 개요: 단일 뉴런의 계산 능력

뇌의 기본 계산 단위인 뉴런은 단순한 "합산 후 발화" 머신이 아니다. 최신 신경과학은 수상돌기(dendrites)가 **비선형 연산**을 수행한다는 것을 발견했다.

**뉴런의 입력 규모:**
- 일반적인 뉴런: 1,000~10,000개 시냅스 입력 (피라미드 뉴런은 30,000개 이상)
- 이들 입력이 시공간적으로 분산되어 도착
- 각 입력은 다른 강도, 부호(흥분/억제), 신경전달물질 종류

**전통적 모델 vs 현실:**

| 측면 | 선형 모델 | 실제 비선형 |
|------|---------|-----------|
| 합산 방식 | EPSP_total = sum(모든 EPSP) | 비선형 증폭 또는 감소 |
| 소뮬탕 타입 | 소뮬탕 = 입력 개수 × 강도 | 위치/시간 패턴 의존 |
| 공간적 계산 | 단일 노드 | 분산된 서브유닛 |
| 동적 범위 | 선형 | 압축/증폭 비선형 |

### 7.2 수상돌기 비선형성(Dendritic Nonlinearity)

#### 초선형(Supralinear) 합산
**조건:** 같은 가지(branch) 내 **근처**의 여러 시냅스가 동시 활성화

**메커니즘:**
- 국소적 탈분극이 전압의존 채널(NMDA, L-type Ca2+ channel) 개방
- 칼슘 유입 증가 → 2차 메신저 급증
- 결과: 단순 합 > 실제 반응

**수식 예:**
```
선형 예측: EPSP_1 + EPSP_2 = 5mV + 5mV = 10mV
실제(초선형): 2 × 7.5mV = 15mV (50% 증폭)
```

**기능적 의미:**
- 동일 수상돌기 부위의 입력 "AND" 게이트처럼 작용
- 특정 패턴 인식 능력 (feature detection)
- 뉴런이 피처 디텍터 역할 가능

#### 아선형(Sublinear) 합산
**조건:** **원거리** 또는 **이종 가지**의 시냅스들, 또는 억제성 입력과 혼합

**메커니즘:**
- 국소 억제로 인한 shunting inhibition
- 시냅스후 전도성 증가 (conductance increase) → 신호 약화
- 케이블 감쇠 (원거리 입력)

**수식 예:**
```
선형 예측: EPSP_1 + EPSP_2 = 5mV + 5mV = 10mV
실제(아선형): 2 × 4mV = 8mV (20% 감소)
```

**기능적 의미:**
- 동적 범위 확대 (압축)
- 잡음 필터링
- 에너지 효율

### 7.3 비선형 계산 모드

#### 모드 1: 전압 의존 NMDA 수용체
- AMPA 채널은 즉각 개방 (빠른 흥분)
- NMDA 채널은 막이 충분히 탈분극되어야만 Mg2+ 차단 해제
- 결과: 협력성(cooperativity) 필요, 여러 입력 필요

#### 모드 2: 백프로퍼게이팅 활동전위(Back-propagating Action Potential)
- 축삭에서 발생한 활동전위가 수상돌기로 역전파
- 수상돌기 칼슘 채널 개방 → Ca2+ 유입 폭주
- 시냅스전 활동(역행 신호) + 시냅스후 활동의 시간적 근접성 감지
- **Spike-Timing-Dependent Plasticity (STDP)** 기초

#### 모드 3: 가지 특이 계산(Branch-Specific Computation)
- 각 수상돌기 가지가 "미니 뉴런" 역할
- 가지별로 다른 게이팅 규칙, 다른 신경조절 환경
- 결과: 단일 뉴런 내 여러 구분 가능한 계산 회로

### 7.4 정보 통합의 실제 규칙

**수상돌기 주요 계산:**

1. **AND 게이트:** 같은 가지, 초선형 → 모든 입력 필요
2. **OR 게이트:** 다른 가지, 선형 → 일부 입력으로 충분
3. **XOR 게이트:** 특정 시간/위치 패턴 필요
4. **NOT 게이트:** 억제성 입력 충분

**계층적 통합:**
```
수상돌기 가지 (계층 1)
├─ 여러 시냅스 → 비선형 계산
├─ 결과: branch output
│
세포체 (축삭 언덕, 계층 2)
├─ 여러 가지 출력 수신
├─ 다시 통합
│
축삭(계층 3)
└─ 최종 이진 결정: 발화 vs 침묵
```

### 7.5 동적 조절

시냅스 통합은 정적이 아니라 **동적**으로 변한다:

**변조 요소들:**
- **신경조절물질(Neuromodulators):** 도파민, 세로토닌, 노르에피네프린 → 채널 감도 변경
- **적응(Adaptation):** 반복 자극 → 시냅스 강도 감소 (presynaptic)
- **촉진(Facilitation):** 빈번한 자극 → 신경전달물질 방출 증가
- **피로(Depression):** 신경전달물질 고갈 또는 수용체 탈감작

### 디지털 구현 핵심 파라미터
- `dendriteSegment[]` = 각 가지 구간
- `segmentVolume = [cubic microns]` [전기 용량성 영향]
- `NMDA_blockage = f(V)` [Mg2+ 의존 블로킹]
- `nonlinearity_factor = β` [0=완전 선형, 1=강한 비선형]
- `subthreshold_conductance_shunt = g_shunt` [억제 전도성]
- `branch_computation_mode = ENUM{AND, OR, XOR, ...}`
- `temporal_window_integration = [ms]` [각 가지별]
- `adaptation_time_constant = τ_adapt`
- `facilitation_time_constant = τ_facil`

## 8. 신호 전달의 시간 스케일

### 8.1 빠른 신호(Fast Signaling): 밀리초(ms) 단위

**활동전위 전파:**
```
시간 프로필:
0 ms      ━━━━━ 안정 상태
0-1 ms    상향: Na+ 내유입 (0 to +30mV)
1-2 ms    재분극: K+ 외유출 (-70mV)
2-3 ms    과분극: 칼슘 채널 개방 준비
3-100ms   상대 불응기: 더 센 자극 필요
```

**시냅스 전달 세부:**
| 단계 | 시간 | 설명 |
|-----|------|------|
| 활동전위 도달 | <1ms | 축삭 말단 탈분극 |
| Ca2+ 채널 개방 | <1ms | 전압 의존 |
| Ca2+ 유입 | 1-2ms | 확산 기반 |
| Exocytosis 유발 | 1-2ms | synaptotagmin-SNARE 상호작용 |
| 신경전달물질 방출 | ~1ms | 소포 융합 |
| 확산 → 수용체 | 1-5ms | 20nm 갭 통과 |
| 수용체 결합 | <1ms | 매우 빠름 |
| 이온채널 개방 | 1-10ms | 컨포메이션 변화 |
| EPSP/IPSP 형성 | 5-20ms | 통합 시간 |

**뉴런-뉴런 지연:**
```
축 A → 축 B 총 지연 = ~1-5ms

구성:
- 축 전도: 매우 빠름 (미엘린이면 <1ms)
- 시냅스 전달: 1-2ms
- 수용체 결합: <1ms
- 채널 개방: 1-5ms
```

### 8.2 느린 신경조절(Slow Neuromodulation): 초(s) ~ 분(min) 단위

**신경조절물질의 특징:**
- 정확하고 빠른 신호 전달이 아니라 **상태 조절**
- 광범위 브로드캐스트 (한 시냅스가 아니라 대뇌 영역 전역)
- 채널 매개변수를 천천히 변경

**메커니즘:**
```
도파민 뉴런
  ↓ (축삭 방출)
광범위 확산 → 도파민 수용체 (D1, D2, ...)
  ↓
G-단백질 신호 (G_s, G_i)
  ↓
2차 메신저 (cAMP, IP3, Ca2+)
  ↓
느린 유전자 발현 변화
또는 채널 인산화 (수 초 ~ 분)
```

**신경조절의 예:**

1. **도파민 신경조절**
   - 시간: 수초 ~ 분
   - 효과: 뉴런 여흥성(excitability) 증감, 학습 신호
   - 광학창(volume transmission): 정확한 시냅스가 아닌 영역 전체

2. **아세틸콜린 (중추)**
   - 시간: 수초 ~ 분
   - 효과: 주의 강화, 학습률 증가
   - 원천: 기저 전뇌(basal forebrain)

3. **칼슘/IP3 신호**
   - 시간: 밀리초 ~ 초
   - 효과: 유전자 전사 조절, 장기 시냅스 변화

### 8.3 시간 스케일 비교 표

| 시간 스케일 | 신호 타입 | 메커니즘 | 기능 |
|-----------|---------|--------|------|
| **<1ms** | 직접 전기적 | 이온 이동 | 신경근 접합부, 긴급 반응 |
| **1-10ms** | 화학 시냅스 | 신경전달물질 확산 | 회로 수준 계산 |
| **10-100ms** | 시냅스 통합 | EPSP/IPSP 합산 | 의사결정 (발화 여부) |
| **100ms-1s** | 국소 회로 | 피드백, 억제 | 리듬 생성, 게이팅 |
| **1-10s** | 신경조절 | G-단백질, 2차 메신저 | 상태 변경, 학습 신호 |
| **분~시간** | 유전자 발현 | 전사 조절 | 시냅스 가소성, 응축 |
| **시간~일** | 단백질 합성 | 신경성장인자 | 장기 메모리, 축제 성장 |

### 8.4 다중 타이밍 수렴(Multiscale Temporal Convergence)

**핵심 통찰:** 뇌는 여러 시간 스케일에서 **동시에** 신호 처리

**예: 두려움 학습**
```
밀리초 (fast)
└─ 청각 신호 (음) + 전기 충격 (통증)
   → 측두엽 → 편도체 뉴런 시냅스

초 (중간)
└─ 동시 활성화된 시냅스들의 EPSP 합산
   → 시냅스 강화 신호 시작 (STDP)

초~분 (느림)
└─ 칼슘 →  cAMP → CaMKII 인산화
   → 수용체 밀도 증가 (AMPA 수용체 삽입)
   → 시냅스 영구 강화

분~시간 (매우 느림)
└─ CREB 인산화 → 즉시초기유전자(IEG) 발현
   → 신경영양인자(BDNF) 생산
   → 시냅스 해부학적 구조 변경
```

이 모든 수준이 조화를 이루어 **배운 연관성(learned association)** 형성.

### 디지털 구현 핵심 파라미터
**Fast Signaling (electrical):**
- `actionPotentialDuration = 1-2 ms`
- `synapticDelay = 0.5-5 ms`
- `dendriteIntegrationWindow = 50-100 ms`

**Slow Signaling (neuromodulation):**
- `neuromodulatorHalfLife = 1-10 s`
- `secondMessengerRiseTime = 100 ms - 1 s`
- `geneExpressionLatency = 30 min - 2 hours`
- `synapticPlasticityWindow = seconds to hours`

**Dual-timescale Neuron State:**
```
struct NeuronState {
  // Fast (electrical)
  V_membrane [mV]
  spike_flag [bool]
  
  // Slow (neuromodulation)
  dopamine_level [nM]
  serotonin_level [nM]
  cAMP_concentration [nM]
  
  // Plasticity (intermediate)
  synapse_strength [0-1] per input
  recent_spike_history [ms window]
  calcium_trace [exponential decay]
}
```

## 9. 역행 신호(Retrograde Signaling)

### 9.1 개요

전통적 신호 전달은 **시냅스전 → 시냅스틈 → 시냅스후**인 "앞 방향"이다. 그러나 뇌는 또한 **역방향** 신호를 사용하여 시냅스를 조절한다.

역행 신호의 특징:
- **방향:** 시냅스후 → 시냅스전 (역방향)
- **메신저:** 가스 또는 지질 분자 (확산 가능)
- **목적:** 시냅스전 신경전달물질 방출 약화 또는 강화
- **기능:** 피드백 안정화, 시냅스 경쟁

### 9.2 엔도카나비노이드(Endocannabinoid) 시스템

**분자들:**
- **아라키도노일에탄올아민(Anandamide, AEA)**
- **2-아라키도노일글리세롤(2-AG)** ← 더 중요함

**형성:**
```
시냅스후 활성화
  ↓
칼슘 유입 (EPSC 수신)
또는 G_q/11 수용체 활성화 (mGluR, M1 무스카린)
  ↓
포스포리파제 C(PLC) 활성화
  ↓
포스파티딜이노시톨(PIP2) → 2-AG + IP3
  (또는 다른 경로로 AEA 형성)
```

**분자 크기:** 아주 작음, 확산 빠름

**시냅스전 효과:**
```
2-AG 또는 AEA
  ↓ (역행 확산)
시냅스전 막
  ↓
CB1 수용체 결합 (G_i/o 타입)
  ↓
cAMP 감소 → PKA 억제
또는 직접 K+ 채널 활성화
  ↓
결과: 신경전달물질 방출 감소 ("음소거" 신호)
```

### 9.3 역행 신호의 시간 스케일과 영속성

**빠른 역행 신호 (Depression, DSI/ISI):**
- **시간:** 수백 ms ~ 초
- **메커니즘:** 시냅스전 Ca2+ 채널 또는 프라이밍 단계 단기 억제
- **회복:** 분 단위

**예: Depolarization-Induced Suppression of Inhibition (DSII)**
```
피라미드 뉴런 발화 (역전파된 활동전위)
  ↓
수상돌기 Ca2+ 급증
  ↓
2-AG 방출
  ↓
GABA 뉴런의 시냅스전 CB1 차단
  ↓
결과: 나를 억제하던 GABA 신경전달물질 방출 감소
       → "자기 억제 해제(disinhibition)"
```

**느린 역행 신호 (장기 가소성):**
- **시간:** 분 ~ 시간
- **메커니즘:** 유전자 발현 또는 단백질 합성 변화
- **분자:** 뉴로트로핀(BDNF), 사이토카인

### 9.4 산화질소(Nitric Oxide, NO) 역행 신호

**특징:**
- **매우 작은 가스 분자** (MW = 30)
- **높은 확산성:** 막을 쉽게 통과
- **짧은 반감기:** 초 단위 (~4s)

**형성 경로:**
```
시냅스후 칼슘 증가
  ↓
NO 합성효소(NOS) 활성화
  ├─ nNOS (neuronal NOS) = 주요 형태
  ├─ eNOS (endothelial NOS)
  └─ iNOS (inducible NOS)
  ↓
L-아르기닌 → NO + L-시트룰린
```

**시냅스전 효과:**
```
NO (역행 확산)
  ↓
시냅스전 guanylyl cyclase (가용성 형태)
  ↓
GTP → cGMP (2차 메신저)
  ↓
cGMP-dependent protein kinase (PKG) 활성화
  ↓
결과:
- synaptophysin 인산화
- 글루타메이트 포함 소포 방출 증강
- 신경전달물질 방출 촉진 ("증강" 신호)
```

**흥미로운 특징:**
- **STDP(Spike-Timing-Dependent Plasticity)의 필수 요소**
- 시냅스전 활동 + 시냅스후 역행 신호의 시간적 근접성 감지
- 학습의 분자 메커니즘 중 일부

### 9.5 역행 신호와 시냅스 경쟁

**상황:**
- 여러 시냅스가 같은 시냅스후 뉴런을 경쟁적으로 자극
- 모든 시냅스가 동일한 강도로는 생존할 수 없음

**역행 신호 메커니즘:**
```
강한 입력 A
  ↓
강한 탈분극 → 많은 역행 신호
  ↓
근처 시냅스 B의 신경전달물질 방출 감소
  ↓
결과: A는 강화, B는 약화 (경쟁 우승)
```

**흥미:** 역행 신호는 **국소적**이다 (몇 미크론). 먼 시냅스는 영향을 덜 받음.

### 디지털 구현 핵심 파라미터
**Endocannabinoid System:**
- `endocannabinoid_concentration` [nM]
- `cb1_receptor_presence = bool` [시냅스전]
- `retrograde_diffusion_distance = 1-10 μm`
- `dsi_duration = 0.5-2 seconds`
- `dsi_recovery_time = 30-60 seconds`
- `basal_release_probability = p_base`
- `suppressed_release_probability = p_base × (1 - endo_effect)`

**Nitric Oxide System:**
- `no_concentration = [nM]`
- `no_halfLife = 4 seconds`
- `nNOS_activity = f(calcium_postsynaptic)`
- `guanylyl_cyclase_activation = f(NO_level)`
- `cGMP_level = [nM]`
- `PKG_phosphorylation_rate = f(cGMP)`
- `facilitation_magnitude = β_NO` (0-100% 증강)
- `STDP_window = ±50 ms` (시냅스 학습)

## 10. 디지털 뉴런 구현: 통합 모델 개요

### 10.1 상태 벡터(State Vector)

```python
class DigitalNeuron:
    # === 막 전위 ===
    V_membrane: float          # mV, 현재 막 전위
    threshold: float = -55     # mV, 활동전위 역치
    rest_potential: float = -70  # mV, 안정 상태
    
    # === 이온 채널 상태 ===
    Na_channel_state: float    # [0, 1], 개방도
    K_channel_state: float     # [0, 1], 개방도
    Ca_channel_state: float    # [0, 1], 개방도 (L-type)
    
    # === 신경전달물질 풀 ===
    vesicles: List[Vesicle]    # 각 시냅스별 소포 재고
    neurotransmitter_reservoir: float  # 재생산용
    
    # === 시냅스 상태 ===
    synapses: Dict[int, Synapse]  # {presynaptic_id: Synapse}
    
    # === 신경조절 상태 ===
    dopamine_local: float      # 국소 도파민 농도 [nM]
    serotonin_local: float     # 국소 세로토닌 농도 [nM]
    
    # === 역행 신호 ===
    endocannabinoid: float     # 시냅스후 2-AG 농도 [nM]
    nitric_oxide: float        # NO 농도 [nM]
    calcium_postsynaptic: float  # 칼슘 추적
    
    # === 적응 상태 ===
    adaptation_current: float  # 누적된 전류
    recent_spike_times: Deque[float]  # 최근 발화 시간들
    
    # === 장기 가소성 ===
    synapse_strengths: Dict[int, float]  # 각 시냅스의 AMPA 수용체 밀도
    cAMP_level: float          # 2차 메신저
    CREB_phosphorylation: float  # 기억 관련 단백질
```

### 10.2 시뮬레이션 루프 (의사코드)

```python
def simulate_neuron(dt=0.001):  # dt = 1ms 타임스텝
    """
    단일 뉴런을 시간 스텝 dt만큼 진행.
    멀티플 타임스케일 통합.
    """
    
    # ===== FAST TIMESCALE (1ms) =====
    # 1. 시냅스 입력 수집
    total_epsc = 0  # 흥분성 전류
    total_ipsc = 0  # 억제성 전류
    
    for synapse_id, synapse in neuron.synapses.items():
        epsc, ipsc = synapse.compute_current(dt)
        total_epsc += epsc
        total_ipsc += ipsc
    
    # 2. 수상돌기 비선형성 적용 (시냅스 통합)
    integrated_input = apply_nonlinear_integration(
        total_epsc, total_ipsc,
        spatial_proximity=True,
        temporal_window=50  # ms
    )
    
    # 3. 막 전위 업데이트 (Hodgkin-Huxley 또는 LIF)
    dV_dt = (
        integrated_input +  # 시냅스 입력
        ion_pump_current +   # Na+/K+ ATPase
        leak_current         # 누출 채널
    ) / membrane_capacitance
    
    neuron.V_membrane += dV_dt * dt
    
    # 4. 채널 상태 업데이트
    update_ion_channels(neuron, dt)
    
    # 5. 활동전위 검사 (All-or-None)
    if neuron.V_membrane >= neuron.threshold and \
       current_time > last_spike_time + absolute_refractory_period:
        neuron.emit_action_potential()
        neuron.last_spike_time = current_time
        
        # 역전파된 활동전위 → 수상돌기 Ca2+ 증가
        neuron.calcium_postsynaptic += Ca_spike_amplitude
    
    # ===== MEDIUM TIMESCALE (100ms) =====
    if time_step % 100 == 0:
        # 시냅스 단기 가소성 (STP)
        for synapse in neuron.synapses.values():
            # Facilitation (신경전달물질 재고 복구)
            synapse.available_nt += recovery_rate * dt
            
            # Depression (재고 고갈)
            if synapse.recently_active:
                synapse.available_nt -= depletion_rate * dt
    
    # ===== SLOW TIMESCALE (초, 분) =====
    if time_step % 1000 == 0:  # 1초마다
        # 역행 신호 업데이트
        neuron.endocannabinoid *= exp(-dt / endo_tau)  # 지수 감소
        neuron.nitric_oxide *= exp(-dt / NO_tau)
        
        # 신경조절 시뮬레이션 (도파민, 세로토닌)
        # 브레인 영역 전체의 신경조절 상태 업데이트
        update_neuromodulator_landscape()
        
        # 신경조절에 따른 채널 민감도 조정
        adjust_channel_sensitivity_by_modulators(neuron)
    
    # ===== VERY SLOW TIMESCALE (시간, 일) =====
    if time_step % 3600000 == 0:  # 1시간마다
        # STDP 기반 장기 가소성 (LTP/LTD)
        for synapse_id, synapse in neuron.synapses.items():
            timing_window = neuron.recent_spike_times[-1] - synapse.last_presynaptic_spike
            
            if abs(timing_window) < STDP_window:
                if timing_window > 0:  # 시냅스전 먼저 → 약화 (LTD)
                    synapse.strength *= (1 - LTD_rate)
                else:  # 시냅스후 먼저 → 강화 (LTP)
                    synapse.strength *= (1 + LTP_rate)
                    # NO/엔도카나비노이드 신호 증가
                    neuron.nitric_oxide += LTP_NO_release
    
    return neuron.V_membrane, neuron.last_spike_time
```

### 10.3 핵심 설계 원칙 (전자두뇌용)

1. **규칙이 아닌 메커니즘 구현**
   - "뉴런은 threshold 넘으면 발화한다" ❌
   - "이온 채널은 전압과 시간에 따라 확률적으로 개방된다" ✓

2. **멀티플 타임스케일 동시 처리**
   - Fast (1ms): 막 전위, 이온 채널, 활동전위
   - Medium (100ms): 시냅스 단기 가소성
   - Slow (초~분): 신경조절
   - Very Slow (시간~일): 유전자 발현, 장기 기억

3. **확률성과 노이즈**
   - 채널 개방/폐쇄: 확률적
   - 신경전달물질 방출: 양자화 (vesicle 단위)
   - 신경조절: 확률적 방출 및 흡수

4. **공간 구조 표현**
   - 수상돌기 세그먼트별 이온 농도 추적
   - 시냅스 위치 의존 비선형성
   - 축삭 미엘린화 영향

5. **피드백 루프**
   - 역행 신호 (엔도카나비노이드, NO)
   - 시냅스 경쟁
   - 신경조절 피드백

## 11. 종합 요약: FlyWire 기반 전자두뇌 구현 시사점

### 핵심 통찰

초파리 뇌(~3,000 뉴런, 완전 커넥톰 지도)로부터 배울 점:

1. **단순성 ≠ 기계성**
   - 초파리도 뉴런 수는 적지만, **세밀한 시냅스 규칙**을 가짐
   - 연결만으로는 부족 → 시냅스 강도, 신경전달물질 타입, 시냅스 위치가 중요

2. **신경전달물질 다양성**
   - 초파리의 3,000개 뉴런에서 최소 20개 이상의 신경전달물질 타입
   - 같은 "연결"이라도 신경전달물질에 따라 완전히 다른 기능

3. **비선형 수상돌기 연산**
   - 뉴런은 단순 가산기 아님
   - 각 수상돌기 가지가 로컬 계산 능력 가짐
   - 행동 창발성을 위해 필수

4. **멀티플 타임스케일**
   - 밀리초(ms): 개별 신경 이벤트
   - 초(s): 회로 다이나믹스 (진동, 게이팅)
   - 분~시간: 학습, 적응
   - 시간~일: 기억 고정화

5. **역행 신호와 국소 피드백**
   - 시냅스전 제어가 완전하지 않음
   - 시냅스후가 피드백을 통해 시냅스전 조절
   - 창발적 안정성 메커니즘
