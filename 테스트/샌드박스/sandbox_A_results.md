## 4. 창발적 행동 분석

### 4.1 관찰된 현상

1. **강한 자극-반응 결합**
   - 모든 자극(P2, P4, P6, P8) 단계에서 운동 뉴런의 강한 활성화
   - 자극 강도와 무관하게 유사한 수준의 운동 반응

2. **상호배제 제어 미약**
   - AVA와 AVB가 모두 동시에 활성화됨
   - 후행(DA+VA)과 전행(DB+VB)의 동등한 활성화
   - 이는 상호억제가 약하거나 차단되어 있음을 시사

3. **네트워크 동작**
   - 감각뉴런 자극 → 국제뉴런 활성화 → 운동뉴런 반응의 체인
   - 빠른 시간 응답 (자극 시작 직후 운동 활성화)

4. **학습 신호**
   - P2와 P8 간 신경반응이 정확히 동일
   - STDP를 통한 시냅스 가중치 변화 미약
   - 네트워크가 반복 자극에 대해 일정한 반응 유지

### 4.2 불일치 분석

실제 C. elegans에서는:
- 접촉(ALM/AVM) → **강한 회피** (후행 > 전행)
- 온도(AFD) → **약한 반응** (회피 또는 적응)
- 유해(ASH/FLP) → **매우 강한 회피**

시뮬레이션에서는:
- 모든 자극에서 후행과 전행이 균등 활성화
- 자극 특이성이 약함
- 상호억제 메커니즘이 제대로 작동하지 않음

### 5.2 개선 필요 사항

× 자극 강도별 차등 반응 부재
× STDP 학습 신호 미약
× 갭접합의 역할 제한적

### 5.3 향후 방향

1. **억제성 시냅스 강화**: 상호억제 가중치 증가
2. **시냅스 특성화**: 자극 특이적 경로 강화
3. **조절 뉴런 추가**: 도파민, 세로토닌 모델링
4. **STDP 파라미터 최적화**: 학습 신호 증폭
