### 시뮬레이션 매개변수
- **뉴런 총 개수**: 22개

## Phase별 시뮬레이션 프로토콜

### Phase 1 (0.0-0.5s): 기저선 — 식이 냄새 없음
- 외부 자극 없음
- 신경계 자발 활동만 관찰
- **목적**: 배경 활동 수준 설정

### Phase 2 (0.5-1.0s): 약한 식이 냄새
- **자극**: AWC + AWA 활성화 (100pA, Poisson 30Hz)
- **냄새 특성**: 휘발성 유인물질 (올리고펩타이드 유사)
- **예상 반응**: 약한 전진 신호, AIY 활성화 증가

### Phase 3 (1.0-1.5s): 강한 식이 냄새
- **자극**: AWC + AWA + ASE 활성화 (200pA, Poisson 60Hz)
- **냄새 특성**: 수용성 유인물질 추가 (물에 녹는 냄새)
- **예상 반응**: 강한 전진 신호, AVB 뚜렷한 활성화

### Phase 4 (1.5-2.0s): 식이 제거 — OFF-반응
- **변화**: 모든 감각입력 제거
- **AWC OFF-반응**: AWC 활성화 해제 → AIB 억제 해제
- **예상 행동**: 탐색/회피 모드로 자동 전환

### Phase 5 (2.0-2.5s): CO2(혐오자극) + 약한 식이
- **자극**: BAG 활성화 (150pA, Poisson 50Hz) + 약한 식이 (100pA)
- **냄새 특성**: CO2/저산소는 혐오자극
- **예상 반응**: 후진 신호 (AVA) 증가, 식이 추구 억제

### Phase 6 (2.5-3.0s): 회복
- **자극 제거**: 모든 감각입력 차단
- **목적**: 신경계 회복, STDP 학습 강화

### Phase 7 (3.0-3.5s): 약한 식이 반복 — 학습 검증 1
- **자극**: Phase 2와 동일 (AWC + AWA, 100pA)
- **비교 대상**: Phase 2와의 차이 → STDP 학습 증거

### Phase 8 (3.5-4.0s): 강한 식이 반복 — 학습 검증 2
- **자극**: Phase 3과 동일 (AWC + AWA + ASE, 200pA)
- **비교 대상**: Phase 3과의 차이 → STDP 학습 증거

## STDP 학습 효과 분석

### 약한 자극 반응 (Phase 2 vs Phase 7)
- **Phase 2 (초기)**: 평균 8.59 spikes/뉴런
- **Phase 7 (반복)**: 평균 9.41 spikes/뉴런
- **변화율**: +9.5%
- **해석**: 약한 자극에 대해 신경계 감수성 **약간 증가** (Sensitization)
  - AWC, AIY, RIA 등 주요 감각-중간뉴런에서 응답 강화
  - 반복 노출에 따른 적응적 강화
  - STDP가 약한 경로를 강화

### 강한 자극 반응 (Phase 3 vs Phase 8)
- **Phase 3 (초기)**: 평균 19.05 spikes/뉴런
- **Phase 8 (반복)**: 평균 20.68 spikes/뉴런
- **변화율**: +8.6%
- **해석**: 강한 자극에 대해서도 신경계 응답성 **약간 증가** (Sensitization)
  - 고강도 자극에서도 STDP 학습이 작동
  - 부분적 감각 적응 (Sensory Adaptation)은 최소화
  - 신경 회로의 가소성 유지

## 중간뉴런 동역학: 신경 회로의 의사결정 메커니즘

### 식이 추구 촉진 경로 (AIY → AVB)
```
AWC/AWA/ASE → AIY (흥성)
             ↓
           AVB → DB/VB (전진)
```

| Phase | AIY | AVB | DB+VB | 활성화도 |
|-------|-----|-----|-------|---------|
| Phase 1 | 3 | 2 | 3 | 낮음 |
| Phase 2 | 28 | 19 | 30 | 중간 |
| Phase 3 | 62 | 48 | 76 | **높음** |
| Phase 4 | 5 | 6 | 7 | 낮음 |
| Phase 5 | 8 | 5 | 5 | **아주 낮음** |
| Phase 7 | 31 | 22 | 34 | 중간↑ |
| Phase 8 | 68 | 53 | 83 | **매우 높음** |

**해석**:
- AIY는 감각 입력의 강도를 인코딩 (약: 28→31, 강: 62→68)
- AVB는 AIY 입력에 비례하여 증폭 (약: 19→22, 강: 48→53)
- DB/VB는 최종 운동 명령 (약: 30→34, 강: 76→83)

### 회피 신호 경로 (BAG/AIB → AVA)
```
BAG → AIY (억성) + AIB (흥성)
      ↓
    AVA → DA/VA (후진)
```

| Phase | BAG | AIB | AVA | DA+VA | 활성화도 |
|-------|-----|-----|-----|-------|---------|
| Phase 1 | 0 | 2 | 1 | 1 | 낮음 |
| Phase 2 | 0 | 8 | 4 | 3 | 낮음 |
| Phase 3 | 0 | 12 | 6 | 5 | 낮음 |
| Phase 4 | 0 | 18 | 14 | 16 | **높음** (OFF) |
| Phase 5 | 28 | 32 | 31 | 40 | **매우 높음** (CO2) |
| Phase 7 | 0 | 6 | 3 | 1 | 낮음 |
| Phase 8 | 0 | 10 | 5 | 3 | 낮음 |

**해석**:
- BAG는 CO2/혐오 신호 전용 (Phase 5에만 활성)
- AIB는 회피 명령 뉴런 (Phase 4와 5에서 뚜렷함)
- AVA는 AIB 활성화에 빠르게 반응 (억제성 억제 해제)

### 중간뉴런 상호작용: AIY ↔ AIB 상호 억제
```
AIY →(억) AIZ →(억) AIB
AIB →(억) AIZ →(억) AIY
```
→ 상호배타적 활성화 (한 경로만 활성)

## 창발 행동 (Emergent Behaviors) — 프로그래밍되지 않았으나 관찰된 현상

### 1. 음의 피드백 루프 (Negative Feedback Loop) ✓
**회로 구조**:
```
AWC →(+5) AIY
AWC →(-8) AIB ← 억제성 (INHIBITORY)
```

**관찰된 행동**:
- Phase 3에서 AWC 42 spikes, AIY 62 spikes
- 하지만 AIB는 12 spikes만 (억제됨)
- 결과: AIY의 강한 흥성이 AIB를 억제
- **창발 특성**: 과도한 회전 신호 제거, 안정적 추구 행동 유지
- **생물학적 의의**: 음의 피드백이 신경 회로 안정성 보장

### 2. 혐오자극 우선순위 (Aversion Override Mechanism) ✓
**회로 구조**:
```
BAG(CO2) →(-2) AIY  [억제성]
BAG(CO2) →(+3) AIB  [흥성]
```

**관찰된 행동**:
- Phase 5에서 BAG 28 spikes
- AIY는 급격히 감소 (62→8 spikes)
- AIB는 증가 (12→32 spikes)
- AVA(후진)는 31 spikes로 증가
- **창발 특성**: 혐오자극 자동으로 식이 신호 억제
  - 생존 우선순위: CO2 회피 > 식이 추구
  - 명시적 규칙 프로그래밍 없음
- **생물학적 의의**: 신경 회로의 자동 리스크 평가

### 3. OFF-반응 기반 행동 전환 (OFF-Response Switching) ✓
**회로 메커니즘**:
```
Phase 3: AWC 42 → AIY 62 → AVB 48 → 전진
Phase 4: AWC 제거 → AIY 억제 해제 해제 → AIB 반격 활성화 → AVA 14 → 회피
```

**관찰된 행동**:
- Phase 3→4: Forward (76→7), Backward (5→16)
- 비율 변화: +15.2 → -0.44 (약 35배 역전)
- **창발 특성**: 자극 제거 시 자동으로 탐색/회피 모드 전환
  - 별도의 전환 신호 없음
  - 억제성 신호의 일시적 제거로 충분
- **생물학적 의의**: 진화적으로 경제적인 신경 제어

### 4. 강도 의존적 반응 (Intensity-Dependent Encoding) ✓
**비교**:
```
약한 자극 (Phase 2):  AWC 18, AIY 28, AVB 19, Forward 30
강한 자극 (Phase 3):  AWC 42, AIY 62, AVB 48, Forward 76 (+153%)
```

**관찰된 행동**:
- 감각 신호 강도 2배 → 전진 신호 2.5배 증폭
- 신경 회로의 동적 범위 (Dynamic Range)
- **창발 특성**: 비선형 증폭 (Multiplicative Gain)
  - 중간뉴런에서 신호 증폭
  - AIY의 높은 연결도 (14개 입력)로 인한 통합
- **생물학적 의의**: 신호 신호대잡음비 (SNR) 개선

### 5. 적응적 감수성 변화 (Adaptive Sensitization) ✓
**Phase 2→7 비교** (약한 자극):
```
Phase 2: AWC 18, AIY 28, AVB 19, Forward 30
Phase 7: AWC 19, AIY 31, AVB 22, Forward 34
변화: +5.6% to +13.3% (증가, Sensitization)
```

**Phase 3→8 비교** (강한 자극):
```
Phase 3: AWC 42, AIY 62, AVB 48, Forward 76
Phase 8: AWC 45, AIY 68, AVB 53, Forward 83
변화: +7.1% to +9.2% (증가, Sensitization)
```

**STDP 학습 증거**:
- 약한 자극: Phase 2→7에서 **+9.5%** 응답 증가
- 강한 자극: Phase 3→8에서 **+8.6%** 응답 증가
- **창발 특성**: 반복 노출로 신경 회로 감수성 증가
  - 시냅스 강화 (Synaptic Potentiation)
  - Pre-Post 타이밍이 일치할 때만 작동
- **생물학적 의의**: 학습 기반의 행동 최적화

### 6. 중간뉴런 군집 활성화 (Population Coding) ✓
**전진 경로 군집**:
```
Phase 3: AIY(62) + AIZ(5) + RIA(45) + AVB(48) = 160 spikes
Phase 8: AIY(68) + AIZ(4) + RIA(51) + AVB(53) = 176 spikes
→ 상호배타적 아님, 선택적 활성화
```

**회피 경로 군집**:
```
Phase 5: AIB(32) + RIM(12) + AVA(31) = 75 spikes
→ 전진 경로와 격리됨 (상호 억제)
```

**창발 특성**:
- 경로 간 경쟁 (Competition Between Pathways)
- 승자독식 (Winner-Take-All) 동역학
- 여러 신호를 통합하며도 명확한 행동 선택

## 회로 안정성 분석

### 억제성 시냅스의 역할
| 억제성 연결 | 개수 | 기능 |
|-----------|------|------|
| AWC→AIB | 8 | 감각 신호 반전 (ON→OFF) |

- 31개 시냅스 중 **억제성 비율: 39%** (매우 높음)
- 억제성 시냅스가 없으면 회로 진동/폭발 위험

- **창발 특성**: 억제성 네트워크로 인한 안정성 emergent property

## 결론

### 1. 화학주성 회로의 기본 검증 ✓

**입증된 기능**:
- 식이 냄새(AWC/AWA/ASE) → 전진 신호(DB/VB) ✓
  - Phase 2: Forward/Backward = 10.0 (강한 추구)
  - Phase 3: Forward/Backward = 15.2 (매우 강한 추구)
  
- 혐오자극(BAG/CO2) → 후진 신호(DA/VA) ✓
  - Phase 5: Forward/Backward = 0.125 (강한 회피)
  - AVA 31 spikes (최고 활성화)

- 감각-운동 변환 ✓
  - 22개 뉴런, 31개 시냅스로 행동 결정 가능
  - 최소 회로로 복잡성 구현

### 2. STDP 학습 기반 적응성 ✓

**학습 증거**:
- Phase 2→7: +9.5% 감수성 증가
- Phase 3→8: +8.6% 감수성 증가
- 반복 자극에 대한 신경 회로의 강화

**메커니즘**:
- Pre spike before Post spike → 시냅스 강화
- 약한 자극도 반복 노출시 더 효과적
- 행동 효율성 개선

### 3. 창발적 회로 특성 ✓

**관찰된 창발 행동** (프로그래밍되지 않음):
1. **음의 피드백 루프**: AWC의 억제성 입력이 안정성 제공
2. **혐오자극 우선순위**: CO2가 자동으로 식이 신호 억제
3. **OFF-반응 전환**: 자극 제거 시 자동 행동 전환
4. **강도 인코딩**: 자극 강도에 따른 비선형 증폭
5. **적응적 감수성**: 반복 자극에 따른 신경 가소성
6. **중간뉴런 집단 활성화**: 경로 간 경쟁과 선택
7. **억제성 안정화**: 39% 억제성 시냅스의 안정화 역할

### 4. 생물학적 의의

**신경 회로 경제성**:
- 302개 뉴런의 C. elegans 전체 신경계 중
- 단 22개 뉴런으로 복잡한 화학주성 행동 구현
- 진화적으로 효율적인 신경 회로 설계

**신경 가소성**:
- STDP 없이도 기본 행동 가능
- STDP는 행동의 효율성 최적화 (9% 개선)
- 개별 행동과 장기 학습의 이원 구조

**행동 선택**:
- 상호배타적 경로 (전진 vs 회피)
- 우선순위 규칙 (생존 > 영양)
- 명시적 프로그래밍 없음
