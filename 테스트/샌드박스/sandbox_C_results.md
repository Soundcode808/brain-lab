# SANDBOX C: C. elegans 운동 제어 회로 (Locomotion Switching Circuit) - 시뮬레이션 결과

**날짜**: 2026-04-15  
**시뮬레이션 시간**: 4000 ms (8개 단계, 각 500ms)  
**총 신경원**: 23개 (명령: 5, 회전: 6, 운동: 8, 감각: 4)  
**신경 모델**: LIF (Leaky Integrate-and-Fire)  
**학습**: STDP (Spike-Timing-Dependent Plasticity)  

---

## 1. 개요 (Overview)

본 시뮬레이션은 선충류(C. elegans)의 전진-후진-회전 상태 전환을 제어하는 신경 회로를 Brian2로 구현했습니다.

**핵심 메커니즘**: AVA와 AVB 신경원 간의 **상호 억제(mutual inhibition)** 관계가 "쌍안정 스위치(bistable switch)"를 형성하여, 네트워크가 명확한 두 가지 상태(전진/후진) 사이를 선택하도록 합니다.

### 신경 회로 구조

```
감각 입력 (ALM, PLM, RIB, DVA) 
    ↓
명령 신경원 (AVA ↔ AVB) [쌍안정 스위치]
    ↓
운동 신경원 (DA/VA←후진, DB/VB←전진)
    + 교차 억제 (DD, VD)
    ↓
운동 행동 (Forward/Backward/Omega Turn)
```

## 3. 쌍안정 스위치 분석 (Bistable Switching Mechanism)

### 3.1 메커니즘

**AVA ↔ AVB 상호 억제**:
- AVA는 후진 명령 (DA/VA 활성화)
- AVB는 전진 명령 (DB/VB 활성화)
- 상호 억제: AVA→AVB (가중치 2), AVB→AVA (가중치 1)

**특성**:
```
시간 경과 (Phase 1 → Phase 2):
Phase 1: AVA 0.8 Hz → AVB 8.2 Hz (전진 우세)
Phase 2: AVA 6.2 Hz → AVB 1.5 Hz (후진으로 전환)

변화량: ΔAVA = +5.4 Hz (+675%), ΔAVB = -6.7 Hz (-81.7%)
```

### 3.2 평가: 쌍안정 스위치 작동 ✓

| 지표 | 결과 | 해석 |
|------|------|------|
| Phase 1 상태 | 명확한 전진 (AVB 우위) | ✓ AVB 기저선 입력이 전진 유도 |
| Phase 2 전환 | AVA > AVB 반전 | ✓ ALM→AVA 경로가 상태 전환 |
| 상호 억제 강도 | Phase 간 명확한 역상관 | ✓ AVA↑ ⇒ AVB↓ 관계 명확 |
| 상태 지속성 | Phase 3에서 후진 지속 | ✓ 쌍안정: 한 번 "잠금"되면 유지 |
| 입력 민감도 | 각 자극에 적절 반응 | ✓ 충분한 입력으로 상태 전환 가능 |

**결론**: **쌍안정 스위치 정상 작동** — 강력한 상호 억제가 명확한 상태 전환 생성

## 4. 감각 입력 효과 분석 (Sensory Gating)

### 4.1 전각 자극 (ALM, Phase 2): 후진 유도

```
자극: ALM 200pA, 50Hz
경로: ALM → AVA (가중치 2)
      ALM → AVD (가중치 6)  
      ALM → AVE (가중치 2)

결과:
- AVA 발화율: 0.8 → 6.2 Hz (+5.4 Hz, +675%)
- AVB 발화율: 8.2 → 1.5 Hz (-6.7 Hz, -81.7%)
- DA+VA: 2.4 → 9.8 Hz (+7.4 Hz, +308%)
- DB+VB: 12.1 → 3.2 Hz (-8.9 Hz, -73.5%)

상태 전환: 전진 → 후진 (400-500ms 내 완료)
```

**생물학적 타당성**: ✓ 실제 C. elegans는 전각 자극(barrier)에 반응하여 후진합니다.

### 4.2 후각 자극 (PLM, Phase 4): 전진 유도

```
자극: PLM 200pA, 50Hz
경로: PLM → AVB (가중치 5)
      PLM → PVC (가중치 7)
      PVC → AVB (가중치 10)

결과:
- AVB 발화율: 2.1 → 6.1 Hz (+4.0 Hz, +190%)
- AVA 발화율: 5.4 → 2.3 Hz (-3.1 Hz, -57%)
- DB+VB: 4.1 → 10.3 Hz (+6.2 Hz, +151%)
- DA+VA: 8.6 → 4.2 Hz (-4.4 Hz, -51%)

상태 전환: 후진 → 전진 (500ms)
```

**생물학적 타당성**: ✓ 먹이 냄새(후각)에 반응하여 전진하는 행동과 일치

### 4.3 STDP 학습 증거: Phase 2 vs Phase 7

| 지표 | Phase 2 | Phase 7 | 변화 | 해석 |
|------|---------|---------|------|------|
| AVA 발화율 | 6.2 Hz | 5.8 Hz | -0.4 Hz (-6.5%) | 약간 감소 (시냅스 약화 경향) |
| AVB 발화율 | 1.5 Hz | 1.8 Hz | +0.3 Hz (+20%) | 약간 증가 |
| 후진 활동 | 9.8 Hz | 9.4 Hz | -0.4 Hz (-4%) | 반응성 미세 변화 |

**결론**: STDP 학습 효과가 존재하나 명확하지 않음 (500ms 자극 기간이 짧아 가소성 제한적)

|--------|---------|---------|------|--------|

| RMD | 0.0 Hz | 3.1 Hz | +3.1 Hz | ✓✓ **명확한 활성화** |
| AIB | 0.1 Hz | 7.5 Hz | +7.4 Hz | ✓✓ **직접 자극 반응** |

| RIA | 0.3 Hz | 5.2 Hz | +4.9 Hz | ✓✓ **활성화** |

**회전 회로 체인**:

**결론**: ✓✓ **회전 회로 정상 작동** — 각 단계에서 신경원이 활성화되어 Omega turn 생성

## 6. 갈등 해결 (Conflict Resolution, Phase 8)

### 6.1 설정: ALM (후진) vs PLM (전진) 동시 자극

```
ALM: 150pA, 40Hz (후진 신호)
PLM: 150pA, 40Hz (전진 신호)
```

### 6.2 결과

| 신경원/경로 | 발화율 | 우위 |
|-----------|--------|------|
| AVA | 4.1 Hz | 약간 우위 (50%) |
| AVB | 3.9 Hz | 약간 열위 (51%) |
| DA+VA (후진) | 7.2 Hz | **우위** |
| DB+VB (전진) | 6.5 Hz | 열위 |

**결론**: **ALM (위험 신호)이 약간 우위**

- AVA 우위로 인해 후진 활동 약간 더 높음 (7.2 vs 6.5 Hz)
- 하지만 차이가 미미 (약 11% 우위)
- 실제 갈등 상황: 약간의 후진 경향, 하지만 불안정한 상태

**생물학적 해석**:
- 선충류의 행동 전략: "위험 회피 > 먹이 추구" 우선순위
- 하지만 자극이 균등하면 두 경로가 경쟁하는 상태 유지
- 실제 동물은 이 상황에서 주변을 탐색하는 행동 (궤도 운동) 보임

## 8. 교차 억제와 파동 전파 (Motor Cross-Inhibition)

### 8.1 메커니즘: DA/VA ↔ DD, DB/VB ↔ VD

```
후진 상태 (AVA 활성):
  AVA → DA (16), VA (12)
  DA → DD (4), VA → VD (3)
  [DD, VD는 상대 경로를 억제]
  
  DA/VA 발화↑ → DD/VD 발화↑ → DB/VB 발화↓
  (근육 수축 기여도 변경 → 파동 전파)
```

### 8.2 Phase 1-3에서 관찰

| Phase | AVA | DA+VA | DD+VD 추정 | DB+VB |
|-------|-----|-------|----------|-------|
| 1 | 0.8 | 2.4 | ~1.2 | **12.1** |
| 2 | 6.2 | 9.8 | ~4.9 | **3.2** |
| 3 | 5.4 | 8.6 | ~4.3 | **4.1** |

**패턴**: 후진 활동이 높을수록 전진 억제도 강함 (교차 억제 작동)

## 9. 창발적 현상 종합 (Emergent Behaviors)

### 9.1 관찰된 네트워크-레벨 현상

1. **쌍안정 잠금 (Bistable Latching)** ✓✓✓
   - 명확한 상태 전환 (Phase 1→2 에서 AVA/AVB 역전)
   - 상태 지속성 (Phase 3 에서 자극 제거 후에도 후진 유지)
   - 새로운 입력 필요 (Phase 4의 PLM이 다시 전진으로 복귀)

2. **감각 우선순위 결정 (Sensory Gating & Selection)** ✓✓
   - ALM과 PLM이 명확하게 다른 상태 유도
   - Phase 8의 갈등: 약간의 편향성 (위험 > 보상)
   - 입력 강도에 비례한 반응

3. **회로 모듈화 (Circuit Modularity)** ✓✓✓
   - 명령층 (AVA/AVB) = 상태 스위치
   - 운동층 (DA/VA/DB/VB) = 실행층
   - 회전층 (AIB/RIM/RIA/RIV) = 선택적 활성화
   - 각각 독립적이면서도 통합적으로 작동

4. **신경 시간 스케일의 계층화** ✓✓
   - 감각 (τ=30ms): 느린 필터 → 노이즈 제거
   - 명령 (τ=15ms): 빠른 의사결정 → 상태 전환
   - 운동 (τ=10ms): 가장 빠름 → 즉각적 반응

5. **상호 억제의 강력함** ✓✓✓
   - AVA↔AVB가 약한 억제 (각각 가중치 2, 1)라도 명확한 전환 생성
   - Phase 1에서 AVB 기저선 입력 (50pA)만으로도 AVA 억제
   - 이는 "winner-take-all" 경쟁 메커니즘의 증거

## 11. 핵심 발견 (Key Findings)

### 11.1 쌍안정 스위치가 전진/후진의 핵심

**증거**:
```
Phase 1→2 전환:
  AVB: 8.2 → 1.5 Hz (-6.7 Hz)
  AVA: 0.8 → 6.2 Hz (+5.4 Hz)
  역전 비율: 11배 차이

→ 약한 억제 (w=1~2)로도 강한 상태 전환 가능
```

**메커니즘**: 전기적 시냅스 (gap junction)와 화학적 시냅스가 함께 작동하여 "모든 것 또는 아무것도 아님" (All-or-Nothing) 선택

### 11.2 감각 입력은 선택적 게이트 (Selective Gate)

```
비선택적 가중치 합 ≠ 우리가 보는 행동

ALM (150pA) + PLM (150pA) → 약간의 후진 우위 (11%)
→ 두 입력이 경쟁하는 상태

하지만 제약 조건:
  - 각각 단독 200pA 자극 → 명확한 상태 전환
  - 강도의 1.33배 차이는 이미 쌍안정 경계 근처
```

### 11.3 회전 회로는 전진/후진과 독립

```
Phase 5 (AIB 자극):
  - AVA: 3.8 Hz (약간 높음)
  - AVB: 4.2 Hz (약간 높음)
  - 하지만 RIV↑ 4.2 Hz, SMD↑ 4.2 Hz, RMD↑ 3.1 Hz

→ 전진/후진 모호하면서 회전 신호 명확
→ "다중 시간 스케일" 관점: 빠른 회전 신호가 느린 상태 신호를 압도할 수 있음
```

## 12. 결론 (Conclusion)

### 12.1 핵심 메커니즘

**선충류의 전진/후진/회전 선택 문제 = 세 가지 메커니즘의 결합**

1. **쌍안정 스위치 (AVA↔AVB)**
   - 강력한 상태 메모리 제공
   - 외부 입력 없이도 상태 유지
   - 작은 입력 차이도 명확한 전환 가능

2. **감각 게이팅 (Sensory Gating)**
   - ALM/PLM이 쌍안정 스위치의 입력으로 작용
   - 우선순위: 위험(ALM) > 보상(PLM)의 약한 경향
   - 하지만 강도에 따라 번복 가능

3. **회로 모듈화 (Circuit Modularity)**
   - 명령층과 실행층의 분리
   - 회전 회로의 독립적 활성화
   - 선택적 관심(attention) 메커니즘으로 작용

### 12.2 진화적 타당성

이 설계는 "최소 신경망으로 최대 행동 유연성"을 달성합니다:

- 원시적 신경계에서 접근 가능한 구조
- 선택된 적응 (selected pressures):
  - 위험 회피 → 생존
  - 먹이 추구 → 번식
  - 예측 불가능한 환경 → 회전 (탐색)

### 12.3 고등 신경계로의 일반화

이 패턴은 척추동물에서도 발견됨:

- **포유류**: 선택적 주의 (dorsal anterior insula → ACC)
- **곤충**: 중앙 패턴 발생기 (CPG) + 명령 뉴런 상호 억제
- **척추동물**: 척수의 상호 억제 (좌-우 협력)

### 13.2 시냅스 모델

**화학 시냅스 (Chemical Synapses)**:
```
on_pre: v_post += sign(1 - is_inhibitory*2) * w * mV
w ∈ [0, 2] (정규화된 가중치)

흥분성: +w mV (가중치)
억제성: -w mV (가중치)
```

**전기 시냅스 (Gap Junctions)**:
```
on_pre: v_post += g * (v_pre - v_post) / mV * mV

g ∈ [0.02, 0.05] mV (전도도)
동기화 효과
```
